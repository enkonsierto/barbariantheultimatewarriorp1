Infinite Energy
Player 1 has infinite energy.
POKE 36220 0 (8D7A): LD (99CB),A > LD (CB),A (tries to write in the ROM zone = does nothing).
POKE 36516 0 (8EA2): LD (99CB),A > LD (CB),A (tries to write in the ROM zone = does nothing).
POKE 39371 200 (p1_energy_gate): If the code was ever patched to NOP, this poke restores RET Z, so P1 never decrements its energy.
Those pokes avoid the self-modify patch that enables the P1 depletion.
