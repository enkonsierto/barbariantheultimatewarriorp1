B2AC: Hit section: attacker frame 32.
Hitbox active when attacker animation frame index = 32. Pairs: (defender_frame, hit_type index into hit_type_table). Terminated by 0xFF.
Overhead slash contact frame. Exclusively uses types 32-35, the only hit types in hit_type_table specific to this section.
Most standing stances react with fall+slash+dmg at near range (type 32). Closer frames at close range (type 33).
fr8 and fr15 react at any distance (type 34: fall+slash/any). fr27 and fr28 at adjacent only (type 35).
fr32 fully blocked (guard2/any, type 6). fr21 has dual entry: fall+slash (type 33) AND guard1/far (type 13).
fr34 deflected at mid-far range (stagger+slash/d1-3, type 15). fr75 and fr81 use guard1/d1-3 (type 16).
Used by hit_detection.
