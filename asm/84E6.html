84E6: Silence routine.
Generates silence when both channels have period 0.
Generates silence for the same duration as a note. Burns CPU cycles without toggling the beeper (port 254). CPL converts the duration so the loop count matches the timing of the main music engine.
silence_routine 84E6 LD A,($8453) Load duration and invert it (CPL) to compute the outer loop count.
84E9 CPL
84EA LD C,A C = outer loop counter (inverted duration).
84EB PUSH BC Preserve BC and AF across the delay loops.
84EC PUSH AF
84ED LD B,$00 B = 0: inner loop runs 256 iterations.
