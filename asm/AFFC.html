AFFC: Hit section: attacker frame 5.
Hitbox active when attacker animation frame index = 5. Pairs: (defender_frame, hit_type index into hit_type_table). Terminated by 0xFF.
Kick contact frame (initial). Most defender stances react with fall+slash+dmg at adjacent range (type 19).
fr5 is blocked (guard2/near, type 14). fr7 is fully blocked (guard2/any, type 6).
fr23 has dual entry: type 19 (fall, adjacent) and type 16 (guard1, d1-3 range).
Crouching and rolling frames (fr50-fr62) react with fall+slash at near or close range (types 20-21).
Used by hit_detection.
