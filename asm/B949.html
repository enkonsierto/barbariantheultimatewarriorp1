B949: P1 keyboard scancode table.
5-byte table: Fire, Up, Down, Left, Right scancodes for P1 (controller type 1).
Written by key_assign key-assignment loop when P1 selects custom keyboard (one_player_mode_3 sets IX=p1_keyboard_table).
Read by human_input_reader each frame when P1 controller type is 1.
The contents of the table are assignments in loading time.
p1_keyboard_table B949 DEFB $22 Right = P (34)
B94A DEFB $25 Up = Q (37)
B94B DEFB $26 Down = A (38)
B94C DEFB $12 Left = I (18)
B94D DEFB $1A Fire = O (26)
