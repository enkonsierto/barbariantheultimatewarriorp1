9B93: Per-frame character update.
Called from combat_loop for each fighter with IY pointing to their state block.
Returns immediately if IY+0=0 (inactive) or frame_counter_hit non-zero (3-frame cycle skip, see frame_counter_hit_inc).
INPUT: IY = character state block. OUTPUT: None. DESTROYS: AF, BC, HL.
char_frame_update 9B93 LD A,(IY+$00) Load fighter block state.
9B96 OR A Return if zero (inactive).
9B97 RET Z
9B98 LD A,($B973) Load frame_counter_hit (hit-detection skip counter)
9B9B OR A Return if non-zero (frame skip).
9B9C RET NZ
9B9D CALL input_gated_moves Process input/movement transition.
9BA0 CALL anim_frame_stepper Advance frame pointer, trigger frame events.
9BA3 JP x_update_and_tackle_gate Tail-call to x_update_and_tackle_gate and returns directly to combat_loop.
