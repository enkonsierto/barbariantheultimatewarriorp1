972B: Game window row-by-row wipe to black.
Used by the routines at main_menu and menu_fade_out.
Clears the game window from top to bottom one tile row per 112 ms step, producing a visible wipe effect. Rows that share columns with the snake HUD are cleared via clear_snake_row, which preserves those columns; all other rows use clear_row, which clears all 28 tiles. The two bottom rows at attribute-map level are cleared separately via clear_attr_row.
INPUT: None. OUTPUT: None. DESTROYS A, BC, HL.
sweep_transition 972B LD HL,$40E2 Set HL to the first tile of the top game window row.
972E LD B,$02 Set loop counter to 2.
sweep_transition_0 9730 PUSH BC Preserve loop counter before call.
9731 CALL clear_snake_row Clear one snake-edge row preserving HUD columns via clear_snake_row.
9734 POP BC Restore loop counter.
9735 DJNZ sweep_transition_0 Repeat for the top 2 snake-edge rows.
9737 LD B,$02 Set loop counter to 2.
sweep_transition_1 9739 PUSH BC
973A CALL clear_row Clear one full 28-tile row and wait 112 ms via clear_row.
973D POP BC
973E DJNZ sweep_transition_1 Repeat for the next 2 full rows.
9740 LD B,$03 Set loop counter to 3.
sweep_transition_2 9742 PUSH BC
9743 CALL clear_snake_row Clear one snake-edge row via clear_snake_row.
9746 POP BC
9747 DJNZ sweep_transition_2 Repeat for the next 3 snake-edge rows.
9749 LD B,$08 Set loop counter to 8.
sweep_transition_3 974B PUSH BC
974C CALL clear_row Clear one full row and wait 112 ms via clear_row.
974F POP BC
9750 DJNZ sweep_transition_3 Repeat for 8 full rows.

975F XOR A A=0, black attribute value.

9763 LD BC,$1601 B=22 (erasable tiles in the last partial row), C=1.
9766 CALL clear_attr_row Clear the last row attributes to black via clear_attr_row.
