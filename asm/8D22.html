8D22: Main Menu loop
Used by the routine at entry_point.
Draws the main menu text and runs the rainbow colour animation loop until a valid option key (0-3) is pressed.
On selection jumps to the corresponding game mode: 0=demo, 1=1P, 2=2P, 3=developer info screen.
INPUT None. OUTPUT None (tail-calls game mode entry points).
DESTROYS AF, BC, DE, HL.
main_menu 8D22 CALL sweep_transition Fade out any previous screen content to black via sweep_transition.
This entry point is used by the routines at one_player_mode, key_assign and two_player_mode.
main_menu_0 8D25 LD HL,$4845 Set HL to the top menu text position 18501.
8D28 LD DE,$BE9B Point DE to the first menu string pointer select_option ("SELECT OPTION").
8D2B CALL draw_string Draw the string via draw_string.
8D2E LD HL,$4886 Advance to next line position 18566.
8D31 INC DE Advance to next string pointer.
8D32 CALL draw_string Draw "0 DEMO".
8D35 LD HL,$48A6 Advance to next line 18598.

8D39 CALL draw_string Draw "1 ONE PLAYER".
8D3C LD HL,$48C6 Advance to next line 18630.
8D3F INC DE Advance to next string.
8D40 CALL draw_string Draw "2 TWO PLAYER".
8D43 CALL rainbow_animation Run the rainbow animation rainbow_animation.
main_menu_1 8D46 CALL play_music Play music and wait for any key via play_music.
8D49 LD A,$23 Test for '0' key scancode 35 via key_state_test.
8D4B CALL key_state_test
8D4E JP NC,menu_fade_out Jump to demo mode menu_fade_out if pressed.

8D53 CALL key_state_test
