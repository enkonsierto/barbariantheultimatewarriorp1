BE9B: Menu String Sequences
Sequences of byte indices into string_pointers_table. Each sequence defines one menu text line built by
concatenating the words at those indices. The byte 255 terminates each sequence.
Used by draw_string (Draws a string line) and main_menu (Main Menu).
select_option BE9B DEFB $05,$06,$FF "SELECT OPTION"
0_demo BE9E DEFB $00,$1C,$FF "0 DEMO" Set to "START GAME" (25) by one_player_mode and two_player_mode at mode selection.
1_one_player BEA1 DEFB $01,$14,$07,$FF "1 ONE PLAYER"
2_two_player BEA5 DEFB $02,$15,$07,$FF "2 TWO PLAYER"
1_sinclair_joystick BEA9 DEFB $01,$16,$09,$FF "1 SINCLAIR JOYSTICK"
2_kempston_joystick BEAD DEFB $02,$17,$09,$FF "2 KEMPSTON JOYSTICK"
3_keyboard BEB1 DEFB $03,$08,$FF "3 KEYBOARD"
select_keys_player BEB4 DEFB $05,$1B,$18,$07 "SELECT KEYS PLAYER"
player_header_1 BEB8 DEFB $01,$FF "1 " Mutable player-number string index; patched to 1 or 2 by 8DF6/8E06 before the key-assignment screen is drawn.
key_for_fire BEBA DEFB $0A,$0B,$0C,$FF "KEY FOR FIRE"
key_for_up BEBE DEFB $0A,$0B,$0D,$FF "KEY FOR UP"
key_for_down BEC2 DEFB $0A,$0B,$0E,$FF "KEY FOR DOWN"
key_for_left BEC6 DEFB $0A,$0B,$0F,$FF "KEY FOR LEFT"
key_for_right BECA DEFB $0A,$0B,$10,$FF "KEY FOR RIGHT"
key_for_pause BECE DEFB $0A,$0B,$11,$FF "KEY FOR PAUSE"
key_for_quit BED2 DEFB $0A,$0B,$12,$FF "KEY FOR QUIT"
player_1_player_2_header BED6 DEFB $18,$18,$18,$07,$01,$18,$18,$07 " PLAYER 1 PLAYER 2 "
BEDE DEFB $02,$FF
1_sinclair_1_sinclair_2 BEE0 DEFB $01,$16,$01,$16,$02,$FF "1 SINCLAIR 1 SINCLAIR 2 "
2_sinclair_keyboard BEE6 DEFB $02,$16,$18,$18,$08,$FF "2 SINCLAIR KEYBOARD"
3_kempston_keyboard BEEC DEFB $03,$17,$18,$18,$08,$FF "3 KEMPSTON KEYBOARD"
4_keyboard_keyboard BEF2 DEFB $04,$08,$18,$18,$08,$FF "4 KEYBOARD KEYBOARD"
BEF8 DEFS $08 Null terminator / end-of-table padding
