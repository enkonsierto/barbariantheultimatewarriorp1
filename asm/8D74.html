8D74: One-player mode selector.
Used by the routine at main_menu.
Entered from main menu main_menu when '1' key pressed.
Replaces "0 DEMO" menu label with "0 START GAME", enables P1 energy loss, sets infinite time, configures P1 human/P2 CPU control.
Then shows controller sub-menu (Sinclair2/Kempston/Keyboard) and jumps to main loop on selection.
INPUT None. OUTPUT None (tails to fadeout or key assignment). DESTROYS AF.
