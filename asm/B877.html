B877: Per-frame overlay tile index table.
One byte per P2 animation frame. Each byte is an index into the P2 sprite bank at other_sprites (tiles 0-52), selecting the overlay tile to draw over the P2 at that frame. Value 255 means no overlay.
Indexed by P2 frame index p2_frame_index. Read by p2_overlay_tile via ADD HL,DE.
