8DFB: P2 controller setup for key-assignment.
Used by the routine at two_player_mode.
Sets P2 controller type to Keyboard and jumps to key_assign to define keyboard.
Called from two_player_mode (2P mode, options 2 and 3: P1 Sinclair + P2 Keyboard, P1 Kempston + P2 Keyboard).
INPUT: None. OUTPUT: None (falls through to key_assign). DESTROYS: AF, IX.
control_setup 8DFB LD IX,$B950 IX=p2_keyboard_table: P2 keyboard scancodes table.
8DFF LD A,$02 Write controller type 2 (P2 keyboard) to P2 state block p2_controller_type.
