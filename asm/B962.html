B962: Input class lookup table.
16-byte table indexed by the lower 4 bits of the key state bitmask.
Converts all 16 possible directional button combinations into one of 9 meaningful input classes (0-8), used as column offsets into the 9-byte movement transition rows at st_free via no_fire_transition_table and fire_transition_table. Read by moves_solver via get_byte_value.
Key bitmask lower 4 bits: bit 0=Right, bit 1=Left, bit 2=Down, bit 3=Up. Left/Right may already be swapped by input_mirror_correcter for mirrored characters.
Index Buttons held Class Meaning
0x0 (none) 0 Stand / neutral
0x1 Right 1 Move towards (forward)
0x2 Left 2 Move away (backward)
0x3 Right+Left 0 Cancel (treated as neutral)
0x4 Down 3 Crouch
0x5 Down+Right 5 Crouch-forward
0x6 Down+Left 7 Crouch-backward
0x7 Down+Right+Left 0 Cancel (treated as neutral)
0x8 Up 4 Jump
0x9 Up+Right 6 Jump-forward
0xA Up+Left 8 Jump-backward
0xB Up+Right+Left 0 Cancel (treated as neutral)
0xC Up+Down 0 Cancel (treated as neutral)
0xD Up+Down+Right 0 Cancel (treated as neutral)
0xE Up+Down+Left 0 Cancel (treated as neutral)
0xF All dirs 0 Full cancel (neutral)
