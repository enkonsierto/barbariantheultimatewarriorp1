95AE: Read byte from address.
Get byte value pointed by HL+A
INPUT: A = offset. HL = base address. OUTPUT: A = Byte value. DESTROYS: HL.
get_byte_value 95AE ADD A,L Add the offset to the low byte of the address.
95AF LD L,A
95B0 JR NC,get_byte_value_0 Handle carry: if addition overflowed, increment the high byte of the address.
95B2 INC H
get_byte_value_0 95B3 LD A,(HL) Read the byte from the calculated address.
95B4 RET Return with the byte value in A.
