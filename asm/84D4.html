84D4: Melody counter still running; check bass counter independently.
Reached when the melody counter (E) has not yet expired. Checks the bass counter (L) independently so both channels can change phase at different times within the same loop iteration.
melody_running 84D4 JR Z,melody_running Dead code, Z is always clear on entry; never branches.
84D6 EX AF,AF' Swaps with alternate registers.
84D7 DEC L Decrements bass counter.
84D8 JP Z,music_engine_3 If L = 0, jumps to music_engine_3 (end of bass half-cycle).
This entry point is used by the routine at music_engine.
melody_running_0 84DB OUT ($FE),A Sends state to beeper.
84DD NOP NOPs for timing adjustment.
84DE NOP
84DF DJNZ music_engine_1 Decrements B and returns to main loop music_engine_1.
84E1 INC C Increments duration counter.
84E2 JP NZ,music_engine_2 If C != 0, continues (jumps to music_engine_2).
84E5 RET Returns.
