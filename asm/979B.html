979B: Clear one game-window row preserving snake edge columns.
Used by the routine at sweep_transition.
Variant of clear_row for tile rows that share pixel columns with the snake border. Iterates 8 scanlines via single_scanline_clear, which skips the leftmost column. For both snake edge columns only the paper bits in the attribute are cleared to black, keeping the ink colour intact. Tail-calls waits_112ms for the 112 ms inter-row pause.
INPUT: HL = VRAM address of the first scanline of the tile row to clear. OUTPUT: HL = VRAM address of the first scanline of the next tile row. DESTROYS: A, B, DE.
clear_snake_row 979B LD B,$08 B=8, one iteration per scanline within the tile row.
979D CALL get_attr_address Derive the attribute page high byte for the leftmost column via get_attr_address.
