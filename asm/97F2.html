97F2: Apply RLE colour attributes to a screen row.
Used by the routine at paints_stage_screen.
Reads the RLE attribute stream from active_bg_attr_ptr and writes colour attributes. The stream is a sequence of count+value byte pairs. Each pair applies one attribute byte to the next <count> consecutive tiles. A count of 255 signals the end of the row.
After the row is processed, active_bg_attr_ptr is advanced past the end-of-row marker, ready for the next row.
INPUT: HL = VRAM address of any scanline within the target screen row. active_bg_attr_ptr active stage attribute stream pointer. OUTPUT: None. DESTROYS: AF, BC, DE, HL, alternate registers.
apply_bg_attr_row 97F2 PUSH HL Pushes the VRAM row address to carry it across the EXX boundary.
97F3 EXX Switches to the alternate register set.
97F4 POP HL Recovers the VRAM row address in the now-active alternate HL.
97F5 CALL calc_prev_scanline_addr Steps back to the top scanline of this tile row via calc_prev_scanline_addr.
97F8 CALL get_attr_address Derives the attribute RAM high byte for this row via get_attr_address.
97FB LD H,A Forms DE as the attribute RAM address for the first tile of this row.
97FC EX DE,HL
97FD LD HL,($B57D) Loads the stream pointer from active_bg_attr_ptr and reads the next stream byte into A.
apply_bg_attr_row_0 9800 LD A,(HL)
Stream format: count+value byte pairs. The first byte of each pair is the number of consecutive tiles to colour; the second is the attribute value to apply. A count of 255 ends the row.
9801 CP $FF End-of-row check; jumps to the pointer update step if the stream byte is 255.
9803 JR Z,apply_bg_attr_row_2
9805 LD B,A Stores the tile run length in B.
9806 INC HL Advances to the attribute value byte of this pair.
9807 LD A,(HL) Reads the attribute value and writes it to the current tile in attribute RAM.
apply_bg_attr_row_1 9808 LD (DE),A
9809 INC DE Advances to the next tile in attribute RAM.
980A DJNZ apply_bg_attr_row_1 Repeats for all tiles in this run.
980C INC HL Advances to the next count+value pair in the stream.
980D LD ($B57D),HL Saves the updated stream pointer to active_bg_attr_ptr.
9810 JR apply_bg_attr_row_0 Loops back to process the next pair.
apply_bg_attr_row_2 9812 INC HL Advances past the end-of-row marker.
9813 LD ($B57D),HL Saves the updated stream pointer to active_bg_attr_ptr.
