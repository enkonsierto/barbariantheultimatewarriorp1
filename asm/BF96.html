BF96: Per-frame horizontal X-delta table
Indexed by IY+8 (current frame index) from x_pos_controller. Each byte is a signed 8-bit displacement added to IY+3 (X tile). Positive values move right, negative values move left. The mirror flag (IY+2 bit 0) negates the value so mirrored characters always move in the correct direction.
x_delta_table BF96 DEFB $00,$01,$01,$01,$00,$00,$00,$00 frames 0-7: walk forward, delta +1 on frames 1-3
BF9E DEFB $00,$00,$00,$00,$00,$00,$02,$01 frames 8-15: enter arena, delta +2 on frame 14, +1 on frame 15
BFA6 DEFB $01,$00,$00,$00,$00,$00,$00,$00 frames 16-23: enter arena cont., delta +1 on frame 16
BFAE DEFB $00,$01,$00,$00,$00,$FE,$FE,$00 frames 24-31: frame 25 +1, frames 29-30 -2 (knockback/rollback)
BFB6 DEFB $00,$FE,$00,$01,$04,$02,$04,$02 frames 32-39: frame 33 -2 (backward step in mv27 death), frame 35 +1, frames 36-39 roll displacement
BFBE DEFB $01,$FF,$FC,$FE,$FC,$FE,$FF,$00 frames 40-47: frame 40 +1, frames 41-46 roll backward (-1,-4,-2,-4,-2,-1)
BFC6 DEFB $00,$00,$00,$01,$FF,$FF,$FF,$FF frames 48-55: frame 51 +1, frames 52-55 -1 (fall/death drift)
BFCE DEFB $00,$00,$00,$00,$00,$00,$00,$00 frames 56-63: no movement
BFD6 DEFB $00,$00,$00,$00,$00,$00,$00,$00 frames 64-71: no movement
BFDE DEFB $00,$00,$00,$FF,$FF,$FE,$00,$00 frames 72-79: frames 75-77 -1/-1/-2 (drag anim)
BFE6 DEFB $FF,$00,$00,$00,$00,$00,$00,$FF frames 80-87: frame 80 -1, frame 87 -1
BFEE DEFB $00,$00,$00,$00,$02,$FE,$01,$00 frames 88-95: frame 92 +2, frame 93 -2, frame 94 +1
BFF6 DEFB $00,$FF,$FF,$FF,$00,$00,$00,$00 frames 96-103: frames 97-99 -1
BFFE DEFB $00,$00 frames 104-105: no movement
