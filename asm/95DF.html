95DF: Sprites mirror checker and correcter.
Used by the routine at sprite_dispatcher.
Check the 12 tiles of a sprite if needs to be mirrored and do it if required.
Mirror correction: for each active tile, 3x RLA on the low byte of DE extracts the tile orientation bit and compares it with IY+2 bit 0 (mirror flag). If they mismatch, bit 6 of the tile index is flipped (XOR 64) and mirror_tile_inplace mirrors the tile pixel data in-place.
INPUT: IX = Current sprite shapes table. HL = frame data pointer (index 1). IY = character state block.
OUTPUT: None. Framebuffer at H=0x5B updated in place. DESTROYS: AF, BC, DE, HL, IX.
sprites_mirror_checker 95DF LD B,$0C B=12 tiles to draw.
sprites_mirror_checker_0 95E1 LD A,(IX+$00) Read slot byte from shapes table via IX+0.
95E4 INC A INC A: test for empty slot (255 wraps to 0).
95E5 JR Z,sprites_mirror_checker_1 Continue to the next tile shape without changing anything else.
Normal path. Check orientation and paint mirrored if required.
95E7 LD A,(HL) Read tile index from (HL). Advance HL past tile index.
95E8 INC HL

95EF AND $01 Store mirror flag bit in C.
95F1 LD C,A
95F2 LD A,(DE) Load low byte of DE (tile orientation byte).
95F3 RLA Shift orientation bit towards bit 0 (RLA x1).
95F4 RLA Shift orientation bit towards bit 0 (RLA x2).
95F5 RLA Shift orientation bit towards bit 0 (RLA x3).
95F6 AND $01 Isolate bit 0: tile orientation flag.
95F8 CP C Compare tile orientation with character mirror flag.
95F9 JR Z,sprites_mirror_checker_1 Orientation matches - tile already correct, skip mirroring.
95FB LD A,(DE) Orientation mismatch - reload tile index.
95FC XOR $40 Flip orientation bit (64) to match it with the new orientation.
95FE LD (DE),A Write corrected tile index back to DE.
