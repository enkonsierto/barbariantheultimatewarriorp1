8477: Plays music (Beeper 2-channel engine).
2-channel music engine for the beeper speaker.
Plays melody (melody_data) and bass (bass_data) simultaneously.
Uses port 254 (bit 4) to generate square waves.
Technique: Channel mixing with independent period counters.
Timing: Uses NOPs and alternate registers for precise timing.
DESTROYS: AF, BC, DE, HL, IX, IY, AF', BC', DE', HL'.

8491 JP C,$853A Dead branch: if note+12 >= 128 would jump into melody data at melody_data_0. The note range 0-40 makes this unreachable; the branch is residual defensive code.
8494 PUSH HL Preserves melody period on stack.
8495 LD HL,$8449 HL = bass buffer address bass_note.
8498 CALL note_to_period_converter Converts bass note to period (call to note_to_period_converter).
849B POP DE Recovers melody period into DE (D=melody).
849C LD A,H Checks if H (bass period) differs from 1.
849D DEC A
849E JR NZ,music_engine_0 If H-1 != 0, jumps to music_engine_0 (notes to play).
84A0 LD A,D Checks if D (melody period) differs from 1.
84A1 DEC A
84A2 JR Z,silence_routine If D-1 = 0, jumps to silence_routine (silence, both channels muted).
