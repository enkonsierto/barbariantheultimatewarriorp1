8ACB: Hit processor, score award and sound dispatcher.
Called once per frame from the game loop. Reads the pending combat event from pending_sound_index.
If 0, returns immediately. Events 5-6 (guard) skip the visual/score logic and jump directly to sound dispatch.
Events 1-4 (hit): draws a hit indicator glyph (6668) near the attacker column if the frame is in hit_indicator_frame_table, awards score if the frame is in score_award_frame_table, then clears pending_sound_index and plays the sound for the event.
Sound index: 1=sound_kick, 2=sound_tackle, 3=sound_slash, 4=sound_head_cut, 5=sound_guard_1, 6=sound_guard_2.
IN None. OUT None. DESTROYS AF, BC, DE, HL, IY.
hit_processor 8ACB LD A,($B6D3) Load pending event code from pending_sound_index. Return if none.
8ACE OR A Return if no event pending.
8ACF RET Z
8AD0 PUSH AF Save event/sound index.
8AD1 CP $05 Guard events 5-6: skip visual and score, jump directly to sound dispatch.
8AD3 JP Z,hit_processor_6
8AD6 CP $06
8AD8 JP Z,hit_processor_6
8ADB LD IY,($B6D4) IY = attacker state block pointer from attacker_block_ptr.
8ADF LD A,(IY+$08) A = current animation frame index (IY+8).
8AE2 LD HL,$B69B Search hit-indicator table hit_indicator_frame_table for this frame. Z=1 if found, DE=row/column data.
8AE5 CALL find_hit_section Call find_hit_section to search the table.
8AE8 JR NZ,hit_processor_4 Frame not in table - skip drawing hit indicator.
8AEA PUSH DE Save DE (row/column data) while computing screen address.
8AEB LD E,(IY+$03) E = attacker X tile column (IY+3).
8AEE LD D,$00 D = 0, DE = 16-bit column index.
8AF0 LD HL,$5940 HL = start of attribute row 22848 (screen row for hit indicator).
8AF3 ADD HL,DE HL += attacker column offset.
8AF4 POP DE Restore DE (row offset in D, original X column in E).
8AF5 LD A,D A = D (row offset). If 0, already on the right row.
8AF6 OR A
8AF7 JR Z,hit_processor_1 Skip row advance if row offset is 0.
8AF9 LD BC,$0020 Advance HL one attribute row (32 bytes) D times.
hit_processor_0 8AFC ADD HL,BC Advance one row.
8AFD DEC A Decrement row counter.
8AFE JR NZ,hit_processor_0 Loop until row offset consumed.
hit_processor_1 8B00 BIT 0,(IY+$02) Test mirror flag (IY+2 bit 0): determines column direction for hit indicator.
8B04 JR Z,hit_processor_2 Not mirrored - column = base + X tile.
8B06 LD C,$08 Mirrored: shift right 8 columns, then subtract X tile (column = base + 8 - X).
8B08 ADD HL,BC HL += 8.
8B09 LD C,E C = X tile column.
8B0A AND A Clear carry before SBC.
8B0B SBC HL,BC HL -= X tile (mirrored column placement).
8B0D JR hit_processor_3 Jump to boundary check.
hit_processor_2 8B0F LD C,E Not mirrored: C = X tile column.
8B10 ADD HL,BC HL += X tile (normal column placement).
hit_processor_3 8B11 LD B,L Save column byte for boundary check (B = L).
8B12 LD A,L
8B13 AND $1F Mask column to 0-31.
8B15 OR A Skip if column is on a boundary tile (0, 1, 30 or 31).
8B16 JR Z,hit_processor_4 Column 0 - skip.
8B18 CP $01 Column 1 - skip.
8B1A JR Z,hit_processor_4
8B1C CP $1F Column 31 - skip.
8B1E JR Z,hit_processor_4
8B20 CP $1E Column 30 - skip.
8B22 JR Z,hit_processor_4
8B24 LD L,B Restore L from saved column byte B.
8B25 LD A,(HL) Align column to tile boundary (AND 248) and add 2-column offset.
8B26 AND $F8
8B28 OR $02 OR 2: place indicator 2 columns into the tile.
8B2A LD (HL),A Store final VRAM pixel address in HL.
8B2B LD ($B6D6),HL Save hit indicator VRAM address to hit_indicator_screen_addr for align_hit_indicator to align each frame.
8B2E LD A,H Convert pixel address H to attribute area: isolate sector bits (AND 3).
8B2F AND $03
8B31 RLCA Shift sector bits left 3 to form attribute page high byte.
8B32 RLCA
8B33 RLCA
8B34 OR $40 OR 64: set attribute page base (0x40xx).
8B36 LD H,A H = attribute address high byte.
8B37 LD A,$3D A = ASCII 61 (hit indicator glyph, UDG index 13).
8B39 CALL draw_character Draw hit indicator glyph at computed screen address via draw_character.
hit_processor_4 8B3C LD A,(IY+$08) Reload frame index (IY+8) for score table lookup.
8B3F LD HL,$B6B7 Search score-award table score_award_frame_table for this frame. Z=1 if found, DE=score delta.
8B42 CALL find_hit_section Call find_hit_section to search the table.
8B45 JR NZ,hit_processor_6 Frame not in score table - jump to sound dispatch.
8B47 LD A,IYl A = attacker character type (IYl).
8B49 CP $59 Is attacker P1 (type 89)?
8B4B JR Z,hit_processor_5 Yes (P1) - jump to P1 score update at 35670.
8B4D LD HL,($B947) P2 path: HL = P2 score buffer p2_score.
8B50 ADD HL,DE Add score delta DE to P2 score.
8B51 LD ($B947),HL Write updated P2 score to p2_score.
8B54 JR hit_processor_6 Jump to sound dispatch.
hit_processor_5 8B56 LD HL,($B945) P1 path: HL = P1 score buffer p1_score.
8B59 ADD HL,DE Add score delta DE to P1 score.
8B5A LD ($B945),HL Write updated P1 score to p1_score.
hit_processor_6 8B5D XOR A Clear pending event code in pending_sound_index.
8B5E LD ($B6D3),A Write 0 to pending_sound_index.
8B61 POP AF Restore event/sound index from stack.
8B62 CP $01 Sound 1 - kick sound_kick.
8B64 JP Z,sound_kick
8B67 CP $02 Sound 2 - tackle sound_tackle.
8B69 JP Z,sound_tackle
8B6C CP $03 Sound 3 - slash sound_slash.
8B6E JP Z,sound_slash
8B71 CP $04 Sound 4 - head cut sound_head_cut.
8B73 JP Z,sound_head_cut
8B76 CP $05 Sound 5 - guard 1 sound_guard_1.
8B78 JP Z,sound_guard_1
8B7B CP $06 Sound 6 - guard 2 sound_guard_2.
8B7D JP Z,sound_guard_2
8B80 RET Return.
