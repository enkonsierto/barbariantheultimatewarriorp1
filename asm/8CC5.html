8CC5: String line renderer.
Used by the routines at main_menu, one_player_mode, key_assign and two_player_mode.
Renders a full line of text composed of multiple words to screen.
For each index, resolves the word address via word_addr_solver and draws it via draw_word. Stops when index value is 255.
INPUT: HL = VRAM destination address. DE = string index list address (select_option).
OUTPUT: None. DESTROYS: HL, DE.
draw_string 8CC5 PUSH DE Preserves the index list pointer.
8CC6 LD A,(DE) Reads the current word index into A.
8CC7 CALL word_addr_solver Resolves word index to character data address in DE.
8CCA CALL draw_word Draws the word. HL advances to the next free column.
8CCD POP DE Restores the index list pointer.
8CCE INC DE Advances to the next index.
8CCF LD A,(DE) Reads the next index.
8CD0 CP $FF Tests for end-of-line sentinel (255).
8CD2 JR NZ,draw_string Loops back if more words remain.
8CD4 RET Return.
