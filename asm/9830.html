9830: Loser X tile position controller
Used by the routine at x_update_and_tackle_gate.
Advances the active character X tile by the per-frame animation delta.
If the character is in an exit animation (mv29, 31, 32 or 33), increments X by 1 per frame until X reaches 30, then tail-calls end_of_fight_dispatcher to end the fight.
For all other animations, applies the signed delta from the frame displacement table, clamping to the play-area boundary (column 23).
INPUT: IY = active character state block.
OUTPUT: IY+3 = updated X tile, or tail-call to end_of_fight_dispatcher when exit animation clears the screen. DESTROYS: AF, B, HL.
