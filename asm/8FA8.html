8FA8: Reset P1 state block for new round.
Called by end_of_fight_dispatcher after P2 death and by 931F after P1 death.
Resets player 1 to its initial combat state at the start of each round. Sets position and energy via pos_energy_reset, then clears the mirror flag so player 1 always starts facing right (towards the P2).
INPUT: None, OUTPUT: None, DESTROYS: AF.
reset_p1_state_block 8FA8 LD IY,$B559 IY = p1_controller_type (player 1 state block). A = 0 as movement ID for pos_energy_reset.
8FAC XOR A
8FAD CALL pos_energy_reset Reset position to tile 0 (left side) and energy to 13 via pos_energy_reset.
8FB0 LD A,($B55B) Read p1_mirror_flags mirror/flags byte, clear bit 0 (mirror flag) so player 1 faces right at round start, write back to p1_mirror_flags.
8FB3 RES 0,A
8FB5 LD ($B55B),A
8FB8 RET Return.
