91C5: Composite tile pointer resolver.
Converts a composite tile index A (0-63) into a pointer BC to its definition byte sequence in BC8D, via the 2-byte pointer table at composite_tile_pointers.
Gets the word offset into composite_tile_pointers, reads the little-endian pointer at that entry and returns it in BC.
INPUT A composite tile index (0-63, derived from control byte AND 0x3F). HL framebuffer address (preserved unchanged). OUTPUT BC pointer to definition bytes in BC8D. DESTROYS AF, BC.
comptile_ptr_resolver 91C5 PUSH HL Saves framebuffer address.
91C6 ADD A,A Computes 2-byte offset into pointer table.
91C7 LD C,A Forms full offset from low byte.
91C8 LD B,$00
91CA LD HL,$BD31
91CD ADD HL,BC Locates table (composite_tile_pointers) entry for this index.
91CE LD C,(HL) Reads pointer into BC.
91CF INC HL
91D0 LD B,(HL)
91D1 POP HL Restores framebuffer address.
91D2 RET Returns with pointer in BC.
