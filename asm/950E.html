950E: Head tile orientation selector.
Checks the facing direction and mirrors it if required.
On a match (Z set), returns immediately. On a mismatch, flips bit 6 of the tile index in the head sprite bank RAM (other_sprites) and mirrors the tile pixel data in-place at that address via mirror_tile_inplace. The caller render_head_tile then renders the updated tile to the framebuffer.
INPUT: DE = tile index address in head sprite bank (resolve_tile_address via render_head_tile). IX+0 = direction flag (head_state): 1=left, 0=right.
OUTPUT: Z set: orientation correct, no action. Z clear: orientation corrected, tile mirrored. DESTROYS AF, B.
head_mirroring 950E LD A,(IX+$00) Map direction flag to bit-6 mask (two RRCA: bit 0 -> bit 6; left=0x40, right=0x00).
9511 RRCA
9512 RRCA
9513 LD B,A B = expected orientation bit.
9514 LD A,(DE) Read tile index; isolate orientation bit (bit 6).
9515 AND $40
9517 CP B Orientation already matches - return Z set.
9518 RET Z
9519 PUSH DE Mismatch - save tile index pointer.
951A LD A,(DE) Reload tile index; flip orientation bit (XOR 0x40).
951B XOR $40
951D LD (DE),A Write corrected index back to head sprite bank RAM.
951E INC E Advance to tile pixel data.
951F CALL mirror_tile_inplace Mirror tile pixel data in-place in the head sprite bank via mirror_tile_inplace.
9522 POP DE Restore tile index pointer.
9523 RET Return Z clear.
