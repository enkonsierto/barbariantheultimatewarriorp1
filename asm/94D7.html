94D7: Render head or P2 overlay tile to framebuffer.
Used by the routines at head_animator and p2_overlay_tile.
Shared tile renderer for the flying head (head_state) and the P2 overlay (p2_overlay_block_state). Reads the tile index from the sequence pointer at IX+2,3, temporarily patches the sprite bank selector at resolve_tile_address_0 to use the head bank (other_sprites), resolves the tile address via resolve_tile_address, then restores the normal bank (barb_sprite_tileset).
Checks tile orientation against the head direction via head_mirroring; on mismatch, flips bit 6 of the tile index and mirrors the tile in-place via mirror_tile_inplace. Renders the result to the framebuffer at IX+6,7 via tile_render_engine.
INPUT: IX = state block (head_state flying head, p2_overlay_block_state P2 overlay); IX+2,3 = tile sequence pointer; IX+6,7 = framebuffer destination.
OUTPUT: None. DESTROYS: AF, BC, DE, HL.
render_head_tile 94D7 LD L,(IX+$02) Load tile sequence pointer low byte from IX+2,3; HL now points to the current tile.
94DA LD H,(IX+$03)
94DD LD A,(HL) Read tile index at current sequence position.
94DE LD HL,$9D00 Load head sprite bank base address (other_sprites).
94E1 LD ($91F9),HL Patch sprite bank operand at 37369 (resolve_tile_address_0); overrides normal bank (barb_sprite_tileset).
94E4 CALL resolve_tile_address Resolve tile address into DE via resolve_tile_address.
94E7 LD HL,$C000 Load normal sprite bank base address (barb_sprite_tileset).
94EA LD ($91F9),HL Restore sprite bank operand at 37369 (resolve_tile_address_0) to normal bank.
94ED CALL head_mirroring Check tile orientation vs head direction via head_mirroring; draw mirrored variant if mismatched.
94F0 LD L,(IX+$06) Load framebuffer destination IX+6,7 into HL.
94F3 LD H,(IX+$07)
94F6 JP tile_render_engine Tail-call tile_render_engine to render the tile.
