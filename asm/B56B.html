B56B: Goblin state block.
IY points here in init_goblin. set_char_moves writes IY+2, IY+4, IY+5, IY+6, IY+8.
IY+0: 0=goblin inactive (cleared by combat_loop when X=29), 255=active (set by init_goblin).
goblin_state_block B56B DEFB $00 IY+0: Active flag (0=inactive, 255=active).
B56C DEFB $00 IY+1: Input state byte. Unused for goblin.
goblin_mirror_flags B56D DEFB $00 IY+2: Mirror and control flags (bit 0=facing direction, bit 7=non-controllable).
goblin_x_pos B56E DEFB $00 IY+3: X tile position. Also used as frame counter by combat_loop.
goblin_frame_ptr B56F DEFW $ADEA IY+4+5: Frame data pointer.
goblin_movement_id B571 DEFB $01 IY+6: Current movement ID (30=goblin entrance).
goblin_energy_byte B572 DEFB $00 IY+7: Energy byte. Unused for goblin.
goblin_frame_index B573 DEFB $00 IY+8: Current frame index byte.
