9BB0: Calculate horizontal distance between fighters.
Used by the routines at x_update_and_tackle_gate and combat_loop.
Reads tile X position of both characters and computes the absolute difference. Result stored in fighter_distance, used by collision_solver to validate hit ranges.
Direction is determined by the mirror flag of P1 (p1_mirror_flags bit 0).
calc_distance 9BB0 LD A,($B55B) Load p1_mirror_flags (P1 flags); test bit 0 (mirror flag).
9BB3 BIT 0,A
9BB5 JR NZ,calc_distance_0
9BB7 LD A,($B55C) Not mirrored: load P1 X (p1_x_pos) into B, load P2 X (p2_x_pos), subtract B.
9BBA LD B,A
9BBB LD A,($B565)
9BBE SUB B
9BBF JR calc_distance_1
calc_distance_0 9BC1 LD A,($B565) Mirrored: load P2 X (p2_x_pos) into B, load P1 X (p1_x_pos), subtract B.
9BC4 LD B,A
9BC5 LD A,($B55C)
9BC8 SUB B
calc_distance_1 9BC9 LD ($B6F2),A Store result (absolute X delta) in fighter_distance.
9BCC RET Return.
