8A82: Switches the active stage graphics and attribute pointers.
Toggles active_bg between 0 and 1, then resolves the corresponding graphics and attribute pointers from bg_bitmap_ptr and bg_attr_ptr, storing them as the active stage for the next background paint.
change_stage_ptr 8A82 LD A,($B576) Load active stage index active_bg, toggle it (XOR 1) and write back.
8A85 XOR $01
8A87 LD ($B576),A
8A8A PUSH AF Preserve toggled index for the second lookup.
8A8B LD HL,$B577 HL = graphics pointer table bg_bitmap_ptr; resolve new graphics pointer via solve_pointer.
8A8E CALL solve_pointer
8A91 POP AF Restore index for the second lookup.
8A92 LD ($B57B),HL Store new graphics pointer in active_bg_bitmap_ptr.
8A95 LD HL,$B57F HL = attributes pointer table bg_attr_ptr; resolve new attributes pointer via solve_pointer.
8A98 CALL solve_pointer
8A9B LD ($B57D),HL Store new attributes pointer in active_bg_attr_ptr.
8A9E RET Return.
