9055: Left-edge AND-mask tile renderer.
Used by the routines at tile_render_engine and tile_renderer.
Renders one scanline of a 3-byte-wide tile. Scans each byte for transparency (0xFF); the first visible byte receives a left-edge column mask via left_mask_gen, and any remaining visible bytes are written directly to the framebuffer.
Called 21 times per tile by the outer loop at left_edge_loop. On return, DE and HL point to the start of the next scanline.
INPUT: DE = tile source address (within a single 256-byte page). HL = framebuffer destination. OUTPUT: None. DESTROYS: AF, C.
left_edge_renderer 9055 LD C,$FF Sets 0xFF as the transparent sentinel.
9057 LD A,(DE) Reads byte 0; if not transparent, applies the left-edge column mask via left_mask_gen; otherwise skips to the byte-1 check.
9058 CP C
9059 JR Z,left_edge_renderer_0
905B CALL left_mask_gen
905E JR left_edge_renderer_2 Byte 0 masked: jumps to write bytes 1 and 2 directly.
left_edge_renderer_0 9060 INC E Byte 0 transparent: reads byte 1; if not transparent, applies the left-edge mask via left_mask_gen; otherwise skips to the byte-2 check.
9061 INC L
9062 LD A,(DE)
9063 CP C
9064 JR Z,left_edge_renderer_1
9066 CALL left_mask_gen
9069 JR left_edge_renderer_3 Byte 1 masked: jumps to write byte 2 directly.
left_edge_renderer_1 906B INC E Bytes 0 and 1 transparent: reads byte 2 and applies the mask if visible; falls through to the scanline advance.
906C INC L
906D LD A,(DE)
906E CP C
906F CALL NZ,left_mask_gen
9072 JR left_edge_renderer_4
left_edge_renderer_2 9074 INC E Writes the remaining visible bytes directly to the framebuffer without masking.
9075 INC L
9076 LD A,(DE)
9077 LD (HL),A
left_edge_renderer_3 9078 INC E
9079 INC L
907A LD A,(DE)
907B LD (HL),A
left_edge_renderer_4 907C PUSH DE Advances the framebuffer destination 30 bytes to the next scanline and steps the source pointer past this scanline.
907D LD DE,$001E
9080 ADD HL,DE
9081 POP DE
9082 INC E
9083 RET
