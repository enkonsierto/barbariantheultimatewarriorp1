B705: P2 head tile sequence.
17-entry tile index sequence for the severed P2 head animation.
Structure identical to p1_head_tile_seq. Tile ID 18 is absent (sequence jumps directly 17->19). Each byte resolved against sprite bank base 40192 by render_head_tile.
Terminated by sentinel 255 at B716, triggering mark_head_landed (head_state_flag = 255).
Tile base: other_sprites. 17 active tile IDs + 1 sentinel = 18 bytes total.
p2_head_tile_seq B705 DEFB $10,$11,$13,$14 Tile IDs: 16,17,19,20 (cycle 1, tile ID 18 absent).
B709 DEFB $10,$11,$13,$14 Tile IDs: 16,17,19,20 (cycle 2).
B70D DEFB $10,$11,$13,$14 Tile IDs: 16,17,19,20 (cycle 3).
B711 DEFB $10,$10,$11,$11,$11 Tile IDs: 16,16,17,17,17 (final phase).
B716 DEFB $FF Sentinel 255: end of sequence, triggers mark_head_landed.
