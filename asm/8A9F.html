8A9F: Post-fight handler. Stage alternation and counter/time reset.
Used by the routines at entry_point and p2_death_score_update.
Called from 8A30 on 2P timeout (timer_flag=0) or from 93A4 on 1P win (timer_flag always 255 during combat). Alternates stage via change_stage_ptr unless fight counter has reached 7. Resets fight counter to 0 and restores time to 90 only on timeout (2P mode). On 1P win (time=255), counter advances normally with no reset.
post_fight_handler 8A9F LD A,($B9C6) Read the fight counter buffer fight_counter.
8AA2 INC A Increment only for comparison - will not be stored.
8AA3 CP $08 Would counter+1 reach 8? (1P fight limit)
8AA5 CALL NZ,change_stage_ptr If not fight 8, alternate the stage via change_stage_ptr.
8AA8 LD A,($B6D8) Load time buffer timer_flag: 255=won by combat, 0=timeout.
8AAB OR A If 0: round ended by timeout (2P mode); if 255: P1 victory by combat.
8AAC JR NZ,post_fight_handler_0 If non-zero (P1 victory), skip the time reset.
8AAE XOR A Timeout path: Reset the fight counter to 0 at fight_counter.
8AAF LD ($B9C6),A
8AB2 LD A,$5A Restore the time to 90 at timer_flag.
8AB4 LD ($B6D8),A
post_fight_handler_0 8AB7 JP entry_point_1 Return to the game loop.
