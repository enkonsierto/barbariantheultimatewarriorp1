B583: Background Attributes.
RLE compressed attributes using pairs of [count, attribute] values. (28x11 chars).
Example: [1,82] means "1 tile with attribute 82".
Each row ends with 255 marker.
Format per row: [count, attr, count, attr, ..., 255]
Graphics bytes are in stage_1_bg for Stage 1 and stage_2_bg for Stage 2.
