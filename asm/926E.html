926E: Main sprite dispatcher.
Used by the routine at combat_loop.
Draws the sprite for the character pointed by IY.
Returns immediately if IY+0 is 0 (inactive character).
Loads frame index from IY+8, resolves frame data pointer from table barb_ptr_table.
Frame data starts with first byte as shape ID, followed by tile indices for the 4x3 sprite grid. Then sets the framebuffer row and calls draws_4_x_3_tiles (4x3 tile renderer).
INPUT: IY = character state block.
DESTROYS: BC, DE, HL, IX.
