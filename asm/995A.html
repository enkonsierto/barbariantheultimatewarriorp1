995A: Snake energy bar renderer.
Used by the routines at entry_point and game_reset.
Blits the current snake animation frame to the HUD energy bar for each character whose draw phase is active (high nibble of IY+7 non-zero).
P1 bar (VRAM 16416): the sprite is mirrored in-place before blitting and restored afterwards, since P1 faces right and needs the flipped orientation.
P2 bar (VRAM 16445): blitted in its natural orientation.
INPUT: None. OUTPUT: None. DESTROYS: IY, AF, BC, DE, HL.
draw_snakes_sprites 995A LD IY,$B559 Select P1 state block. (p1_controller_type)
995E LD A,(IY+$07) Load P1 packed energy/phase byte.
9961 AND $F0 Isolate draw phase index.
9963 JR Z,draw_snakes_sprites_0 Phase idle - skip P1 bar update.
9965 CALL update_snake_frame_id Advance draw phase and fetch snake sprite pointer via update_snake_frame_id.
9968 PUSH DE Save sprite pointer for the restore pass.
9969 PUSH DE Save sprite pointer for the blit pass.
996A LD B,$1E Set scanline count for the mirror operation.
996C CALL mirror_tile_inplace_0 Mirror snake sprite in-place via mirror_tile_inplace_0.
996F POP DE Restore pointer to the now-mirrored sprite data.
9970 LD HL,$4020 Set P1 energy bar VRAM destination.
9973 LD BC,$031E Set blit dimensions - 3 bytes wide, 30 scanlines.
9976 CALL copy_rect Blit mirrored sprite to P1 energy bar via copy_rect.
9979 POP DE Restore pointer to the mirrored sprite data.
997A LD B,$1E Set scanline count for the restore pass.
997C CALL mirror_tile_inplace_0 Un-mirror sprite in-place via mirror_tile_inplace_0, restoring original data for reuse.
draw_snakes_sprites_0 997F LD IY,$B562 Select P2 state block.
9983 LD A,(IY+$07) Load P2 packed energy/phase byte.
9986 AND $F0 Isolate draw phase index.
9988 RET Z Phase idle - return.
9989 CALL update_snake_frame_id Advance draw phase and fetch snake sprite pointer via update_snake_frame_id.
998C LD HL,$403D Set P2 energy bar VRAM destination.
998F LD BC,$031E Set blit dimensions - 3 bytes wide, 30 scanlines.
9992 JP copy_rect Tail-call to blit sprite to P2 energy bar via copy_rect.
