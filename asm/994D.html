994D: Read movement properties byte.
Used by the routines at set_char_moves and combat_loop.
Reads the properties byte for the given movement ID from the table at AED3.
INPUT: A = movement ID. OUTPUT: A = properties byte. DESTROYS: None.
get_moves_properties 994D PUSH HL Preserve HL and DE.
994E PUSH DE
994F LD HL,$AED3 Load movement properties table base AED3.
9952 LD E,A Form a 16-bit offset from the movement ID.
9953 LD D,$00
