B95A: Sinclair Interface 2 port 1 fixed scancode table.
5-byte read-only table for controller type 4 (Sinclair Interface 1, keys 1-5).
Values: 4,12,20,36,28 (keyboard keys 5,4,3,1,2).
Never written at runtime. Read by human_input_reader when P2 controller type is 4
(2P option 1: Sinclair2 + Sinclair1).
sinclair_joystick_1_table B95A DEFB $04 Fire = key 5
B95B DEFB $0C Up = key 4
B95C DEFB $14 Down = key 3
B95D DEFB $24 Left = key 1
B95E DEFB $1C Right = key 2
