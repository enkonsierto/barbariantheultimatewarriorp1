8D09: Time or fight counter display updater.
Used by the routines at entry_point and timer_tick.
Redraws the 2-digit time HUD every frame.
If timer_flag is 255 (infinite time, 1P/demo mode), displays the fight counter from fight_counter instead.
INPUT: None. OUTPUT: None. DESTROYS: AF, BC, DE, HL, IX.
update_time_display 8D09 LD A,($B6D8) Loads the round timer from timer_flag.
8D0C CP $FF Tests for 255 (infinite time mode).
8D0E JR NZ,update_time_display_0 Non-255: uses the timer value directly.
8D10 LD A,($B9C6) 255: loads the fight counter from fight_counter instead.
update_time_display_0 8D13 LD HL,$40AF Sets HL to the 2-digit timer HUD tile address (16559).
8D16 LD E,A Places the value into DE with high byte 0.
8D17 LD D,$00
8D19 LD B,$02 Sets B=2 for two digits.
8D1B LD IX,$B9A4 Points IX to the 2-digit divider table B9A4.
8D1F JP print_dec_number_0 Tail-calls the decimal printer at print_dec_number_0.
