95A0: Generic 16-bit pointer lookup.
Used by the routines at change_stage_ptr, character_ai_engine, word_addr_solver, set_char_moves, sprite_dispatcher, moves_solver and update_snake_frame_id.
Reads a pointer from a table given a index in A. Adds A*2 to HL and reads the 2-byte entry.
INPUT: A = index. HL = table base address. OUTPUT: HL = resolved pointer. DESTROYS: AF, DE, HL.
solve_pointer 95A0 PUSH DE Save table base (HL) on the stack.
95A1 PUSH HL
95A2 LD L,A HL = A * 2 (byte offset into the pointer table; each entry is 2 bytes).
95A3 LD H,$00
95A5 ADD HL,HL
95A6 POP DE DE = table base.
95A7 ADD HL,DE HL = table base + offset, points to the target entry.
95A8 LD E,(HL) DE = 16-bit pointer from table entry.
95A9 INC HL
95AA LD D,(HL)
95AB POP HL Restore HL (table base).
95AC EX DE,HL Swap DE and HL: HL = resolved pointer.
95AD RET Return.
