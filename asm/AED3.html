AED3: Movement flags table
One byte per movement ID (0-39). Bit flags controlling combat activation, interruptibility and draw order.
Bit 7: combat-active (1=in-combat movement);
Bit 6: non-interruptible (input ignored);
Bit 0: draw on top (when overlapping).
Bits 1-5 always 0. Read on movement change for state flags (IY+2 bit 7) and render priority.
