9127: Renders a 3-byte-wide sprite tile over 21 scanlines using AND masking.
Per scanline: saves 3 framebuffer bytes to AND-mask scratch (mask_scratch_buffer), loads mask from tile data via left_edge_renderer, draws sprite pixels via right_edge_renderer, then ANDs with scratch mask for transparency.
Framebuffer/tile pointers use alternate register bank (EXX) so right_edge_renderer auto-advances to next scanline.
INPUT: B = scanlines (21), HL = framebuffer address, DE = tile data pointer.
OUTPUT: None. DESTROYS: AF,BC,DE,HL (and shadows).
tile_renderer 9127 LD B,$15 B = 21 (full tile height).
This entry point is used by the routine at comp_tile_dispatcher.
tile_renderer_0 9129 PUSH BC save B (scanlines remaining).
912A PUSH HL save HL (framebuffer), DE (tile ptr).
912B PUSH DE
912C EXX EXX: framebuffer HL/tile DE -> alternate registers.
912D POP DE DE = tile data (recovered).
912E POP HL HL = framebuffer (recovered).
912F PUSH HL re-save HL for AND step (914F).
9130 LD A,(HL) save 3 background bytes to AND-mask scratch (mask_scratch_buffer).
9131 LD ($B95F),A
9134 INC L
9135 LD A,(HL)
9136 LD ($B960),A
9139 INC L
913A LD A,(HL) A = framebuffer byte 2
913B LD ($B961),A A save to scratch buffer+2 (mask_scratch_buffer+2)
913E DEC L DEC L x2: restore HL to start of 3-byte sequence
913F DEC L
9140 EXX EXX: back to main registers. HL'/DE' hold framebuffer/tile pointers.
9141 LD HL,$B95F HL = mask scratch buffer (mask_scratch_buffer).
9144 CALL left_edge_renderer left_edge_renderer: load AND mask from tile data into scratch buffer.
9147 EXX EXX: activate HL=framebuffer, DE=tile for right_edge_renderer.
9148 CALL right_edge_renderer right_edge_renderer: draw one scanline of sprite pixels to framebuffer.
914B EXX EXX: back to main registers (now point to next scanline).
914C LD HL,$B95F HL = scratch buffer (reload).
914F POP DE DE = framebuffer address of the scanline just rendered.
9150 LD A,(DE) framebuffer byte 0 (w/sprite); AND scratch byte 0 (mask); restore transparent areas
9151 AND (HL)
9152 LD (DE),A
9153 INC HL next byte
9154 INC E
9155 LD A,(DE) framebuffer byte 1; AND scratch byte 1; restore transparent areas
9156 AND (HL)
9157 LD (DE),A
9158 INC HL next byte
9159 INC E
915A LD A,(DE) framebuffer byte 2; AND scratch byte 2; restore transparent areas
915B AND (HL)
915C LD (DE),A
915D EXX restore HL=framebuffer, DE=tile ptr (next scanline)
915E POP BC recover B=scanlines remaining
915F DJNZ tile_renderer_0 copied one scanline. Repeat for all scanlines
9161 RET tile rendering complete
