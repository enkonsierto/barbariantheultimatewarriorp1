9827: Copy one 28-byte scanline from stage background to screen.
Copies exactly 28 bytes from the stage background source at DE to the screen at HL, advancing both pointers. Called 8 times per tile row by copy_bg_row.
INPUT: DE = stage background source address. HL = VRAM destination address.
OUTPUT: DE and HL both advanced by 28 bytes. DESTROYS AF, B.
copy_bg_scanline 9827 LD B,$1C Sets the tile column counter to 28.
copy_bg_scanline_0 9829 LD A,(DE) Reads one background byte and writes it to VRAM.
982A LD (HL),A
982B INC HL Advances both source and destination pointers.
982C INC DE
982D DJNZ copy_bg_scanline_0 Repeats for all 28 tile columns.
982F RET Returns to copy_bg_row.
