9322: Goblin entry trigger.
Called from anim_frame_stepper when the dying character's animation reaches frame index 20 or 33, signalling the body has settled and the goblin can enter.
Two guard conditions are checked before spawning: the goblin is already active (R46443=255), or the severed head is still in flight (head_state_flag neither 0 nor 255). If no decapitation occurred (R46810=0) or the head has already landed (head_state_flag=255), both guards pass and the goblin is spawned via init_goblin.
INPUT: IY = active character state block, preserved across the call.
OUTPUT: None. Side effect: goblin state block initialised and goblin_state_block set to 255 if spawn conditions are met.
DESTROYS: AF.
goblin_entry 9322 LD A,($B56B) Skip if the goblin is already active (goblin_state_block=255).
9325 CP $FF
9327 RET Z
9328 LD A,($B6DA) Head landed or no decapitation(head_state_flag=255): proceed to spawn.
932B CP $FF
932D JR Z,goblin_entry_0
932F OR A Head still in flight (head_state_flag!=255): return and wait.
9330 RET NZ
goblin_entry_0 9331 PUSH IY Save the caller's IY before goblin initialisation overwrites it.
