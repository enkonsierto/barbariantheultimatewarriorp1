9BCD: Beeper sound engine.
Used by the routine at sound_tackle.
Plays a sequence of square-wave tones from a 0-terminated period table pointed by IX. For each non-zero period byte, emits one complete HIGH/LOW cycle via beeper_emit, then advances to the next byte.
Called by all combat sound dispatchers: sound_tackle (tackle), sound_kick (kick), sound_slash (slash), sound_head_cut (head cut), sound_guard_1 (guard 1), sound_guard_2 (guard 2), sound_round_end (round end).
INPUT: IX = base address of sound data table (period bytes, 0-terminated). OUTPUT: None. DESTROYS: AF, BC, IX.
beeper_engine 9BCD LD A,(IX+$00) Load next period byte from IX. Return if zero (end of sequence).
9BD0 OR A
9BD1 RET Z
9BD2 CALL beeper_emit Emit one square-wave cycle with this period via beeper_emit.
9BD5 INC IX Advance IX to next period byte and repeat.
9BD7 JR beeper_engine
