92A0: Animation frame stepper.
Used by the routines at combat_loop and char_frame_update.
Advances the animation frame pointer for the character pointed by IY, fires events for specific frame indices, and processes end-of-sequence sentinels.
Frames 20 and 33 trigger a goblin spawn check via goblin_entry. Frame 87 (dying) plays the tackle sound via sound_tackle (The winner kicks the loser). Sentinel 253 restarts the current movement via set_char_moves, 254 holds on the last frame, and 255 chains to the next queued animation.
INPUT: IY character state block.
OUTPUT: IY+4,IY+5 updated frame pointer. IY+8 new frame index.
DESTROYS: AF, DE.
anim_frame_stepper 92A0 LD E,(IY+$04) Load frame pointer from IY+4/IY+5 into DE.
92A3 LD D,(IY+$05)
92A6 LD A,(DE) Read current byte.
92A7 CP $14 Compare with 20.
92A9 CALL Z,goblin_entry Frame 20 triggers goblin spawn check via goblin_entry.
92AC INC DE Increment DE (advance frame pointer)
92AD LD (IY+$04),E Store back in IY+4/IY+5
92B0 LD (IY+$05),D
92B3 LD A,(DE) Read new byte into A
92B4 LD (IY+$08),A Store in IY+8.
92B7 CP $21 Compare A with 33.
92B9 PUSH AF Frame 33 also triggers goblin spawn check via goblin_entry, preserving AF.
92BA CALL Z,goblin_entry
92BD POP AF
92BE INC DE Increment DE.
92BF PUSH AF
92C0 CP $57 Compare A with 87.
92C2 PUSH DE
92C3 PUSH AF
92C4 CALL Z,sound_tackle if frame 87 (dying), call sound_tackle (loser receives a kick).
92C7 POP AF
92C8 POP DE
92C9 LD A,(DE) Read updated byte from DE.
92CA CP $FD Compare with 253 (end-of-sequence sentinel).
92CC JR C,anim_frame_stepper_0 If <253, jump to anim_frame_stepper_0 (normal frame, set RES 7,IY+2).
92CE CALL anim_frame_stepper_3 Call anim_frame_stepper_3 (clear bit 7 of IY+2: non-controllable flag off).
anim_frame_stepper_0 92D1 DEC DE Decrement DE.
92D2 POP AF Recover AF from stack.
92D3 CP $FD If A<253, return (normal case).
92D5 RET C
92D6 CP $FD If A=253, Jump forward to anim_frame_stepper_1 (load IY+6 (movement ID) and call set_char_moves_0 to apply movement loop).
92D8 JR Z,anim_frame_stepper_1
92DA CP $FF If A=255 concatenates next animation (call anim_frame_stepper_4 to reset: INC DE, jump to anim_frame_stepper_2).
92DC JR Z,anim_frame_stepper_4
92DE DEC DE If else (A=254): restore frame pointer to DE-1;
92DF LD (IY+$04),E Update IY+4/IY+5 and IY+8.
92E2 LD (IY+$05),D
92E5 LD A,(DE)
92E6 LD (IY+$08),A
92E9 JR anim_frame_stepper_3 And jump to anim_frame_stepper_3.
anim_frame_stepper_1 92EB LD A,(IY+$06) If A=253, (load IY+6 (movement ID) and call set_char_moves_0 to apply movement loop).
92EE CALL set_char_moves_0
92F1 JR anim_frame_stepper_3 Reset IY+2 and return.
anim_frame_stepper_2 92F3 CALL set_char_moves (A=255) Call set_char_moves (to reload next animation).
anim_frame_stepper_3 92F6 RES 7,(IY+$02) RES 7,IY+2; return (clears the non-controllable bit).
92FA RET
anim_frame_stepper_4 92FB INC DE INC DE; read byte; jump to anim_frame_stepper_2.
92FC LD A,(DE)
92FD JR anim_frame_stepper_2
