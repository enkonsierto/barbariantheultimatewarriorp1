8448: Music engine state buffers.
12-byte working area for the 2-channel beeper music engine. Used by music_engine.
The loop-pointer slots at melody_loop_ptr and bass_loop_ptr are unused pointers where play_music would store a fixed restart address. Superseded by restart_music, which reads the restart address inline from the music data stream.
melody_note 8448 DEFB $00 Current melody note index; used by note_to_period_converter to look up period in note_table
bass_note 8449 DEFB $00 Current bass note index; used by note_to_period_converter
beeper_state 844A DEFB $00 Speaker output bit state, 0 or 16; toggled each half-cycle by music_engine
melody_ptr 844B DEFW $EA60 Current read position in melody sequence melody_data; updated by select_music_pointer
melody_loop_ptr 844D DEFW melody_data_0 - UNUSED. Melody loop-back address; never written in the final build
bass_ptr 844F DEFW $EE48 Current read position in bass sequence bass_data; updated by select_music_pointer
bass_loop_ptr 8451 DEFW $8770 - UNUSED. Bass loop-back address; never written in the final build
note_duration 8453 DEFB $F0 Duration counter for the current note; loaded each note by music_engine
