89A8: Game Entry Point
Sets up the system and runs the menu.
entry_point 89A8 DI Game Loaded (Entry Point)
89A9 LD SP,$0000 Initialises the stack pointer to 0.
89AC CALL game_start Jumps to the game start routine.
This entry point is used by the routine at game_reset.
entry_point_0 89AF DI Disable interrupts.
89B0 LD SP,$0000 Initialises the stack pointer to 0.
89B3 LD A,$FF Write 255 to the fight counter buffer. (fight_counter).
89B5 LD ($B9C6),A
89B8 CALL main_menu Runs the main menu via main_menu.
89BB LD HL,$0000 Clears the stack and zeroes, p1_score and p2_score. (Scores Buffer).
89BE PUSH HL
89BF LD ($B945),HL
89C2 LD ($B947),HL
89C5 CALL paint_score Resets and draws the Player 1 score HUD via paint_score.
89C8 CALL paint_score_0 Resets and draws the Player 2 score HUD via paint_score_0.
89CB LD HL,$40AF Sets the time HUD screen address.
89CE POP DE Loads DE = 0 for the initial time value.
89CF LD B,$02 B = 2 two digits to print.
89D1 LD IX,$B9A4 Points IX to the time value buffer B9A4.
89D5 CALL print_dec_number_0 Converts the time value to ASCII and draws it via R36063.
89D8 CALL set_infinite_time Sets the round timer mode via set_infinite_time.
This entry point is used by the routine at post_fight_handler.
entry_point_1 89DB XOR A A = 0.
89DC LD ($B972),A Clears isr_counter, head_state_flag and goblin_state_block to 0.
89DF LD ($B6DA),A
89E2 LD ($B56B),A
89E5 CALL copy_bg_to_frmbuff Copies the stage 1 background to the screen buffer via copy_bg_to_frmbuff.
89E8 CALL paints_stage_screen Paints the stage to the screen row by row via R38849.
89EB CALL reset_p1_state_block Resets the P1 state block via reset_p1_state_block tile 0, full energy, facing right.
89EE CALL reset_p2_state_block Resets the P2 state block via reset_p2_state_block tile 23, full energy, facing left.
This entry point is used by the routine at pause_handler_routine.
game_loop 89F1 CALL copy_bg_to_frmbuff Copy active stage background to screen buffer via copy_bg_to_frmbuff.
89F4 CALL combat_loop Main character update and render dispatch (both fighters + goblin) via combat_loop.
89F7 CALL head_animator Severed head physics update and rendering via head_animator.
89FA CALL draw_snakes_sprites Snake energy bar animation update via draw_snakes_sprites.
89FD CALL tackle_side_swap Per-frame movement auto-update via tackle_side_swap.
8A00 EI Enable interrupts.
8A01 HALT Wait for 50Hz frame interrupt (frame sync).
8A02 DI Disable interrupts.
8A03 CALL frmbuf_to_scr Copy screen buffer to video RAM via frmbuf_to_scr.
8A06 CALL align_hit_indicator Align hit indicator tile address via align_hit_indicator.
8A09 CALL frame_counter_hit_inc Per-frame screen effect update via frame_counter_hit_inc.
8A0C CALL hit_detection Per-frame hit collision detection between fighters via hit_detection.
8A0F CALL hit_processor Hit processor: score award and sound dispatch via hit_processor.
8A12 CALL paint_score Refresh player 1 score display via paint_score.
8A15 CALL paint_score_0 Refresh player 2 score display via paint_score_0.
8A18 CALL update_time_display Update time / fight counter HUD via update_time_display.
8A1B LD A,($B94F) Load quit key scancode from quit_scancode.
8A1E CALL key_state_test Test if quit key is pressed via key_state_test.
8A21 JR NC,game_reset Quit pressed (carry clear): jump to new round reset at game_reset.
8A23 LD A,($B94E) Load pause key scancode from pause_scancode.
8A26 CALL key_state_test Test if pause key is pressed via key_state_test.

8A2F OR A Test if zero.
8A30 JP Z,post_fight_handler Timer = 0 (2P timeout): jump to post-fight handler at post_fight_handler.
8A33 JR game_loop Loop back to next frame at game_loop.
