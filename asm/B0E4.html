B0E4: Hit section: attacker frame 12.
Hitbox active when attacker animation frame index = 12. Pairs: (defender_frame, hit_type index into hit_type_table). Terminated by 0xFF.
Roll/tackle contact frame. Most stances react with stagger+slash+dmg (types 25-28, range variants).
fr12 fully blocked (guard2/any, type 6). fr57 also fully blocked (guard2/any, type 6).
fr40 special case: fall+kick+dmg/near (type 8) - jumping defender is knocked down harder.
Range narrows on crouching frames (fr21-fr23: close, fr24-fr26: any range).
Used by hit_detection.
