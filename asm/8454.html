8454: Selects the next pointer and the note.
Reads the next note from a music sequence and updates the pointer.
If it finds the end marker (64), it restarts the sequence in a loop.
INPUT: HL = address of sequence pointer (melody_ptr or bass_ptr).
OUTPUT: HL = next pointer, A = note value.
DESTROYS: DE.
select_music_pointer 8454 LD E,(HL) Loads the current note address into DE (low and high byte).
8455 INC HL
8456 LD D,(HL)
8457 INC DE DE now points to the next note.
This entry point is used by the routine at restart_music.
select_music_pointer_0 8458 LD A,(DE) Reads the note value into A.
8459 CP $40 Compares with 64 (end of sequence marker).
845B JR Z,restart_music If 64, jumps to restart_music to restart.
845D LD (HL),D Saves the new pointer (updates position in sequence).
845E DEC HL
845F LD (HL),E
8460 RET Returns with note in A.
