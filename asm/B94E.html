B94E: Pause key Scancode Buffer
Pause Scancode Buffer only configurable when choosing keyboard.
Cleared to 0 by one_player_mode_3 and two_player_mode_2/two_player_mode_3 before key-assignment.
Captured by key_assign only if 0 (not yet assigned).
Read by entry_point (game loop) and pause_handler_routine (pause handler) via key_state_test.
pause_scancode B94E DEFB $18 Pause key scancode. 0=unassigned.
