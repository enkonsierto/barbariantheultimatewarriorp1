95C4: Mirrors a tile horizontally in-place.
For each scanline: reverses bits of the 3 bytes via table at reversal_table.
Called from sprites_mirror_checker, draw_snakes_sprites and head_mirroring.
INPUT: B = scanline count. DE = tile address.
OUTPUT: DE advanced past tile. Tile mirrored in-place. DESTROYS: AF, BC, HL.
mirror_tile_inplace 95C4 LD B,$15 B=21h: full tile height.
This entry point is used by the routine at draw_snakes_sprites.
mirror_tile_inplace_0 95C6 LD H,$5B HL base = bit-reversal table (reversal_table).
mirror_tile_inplace_1 95C8 LD A,(DE) A = byte 0 index from tile.
95C9 LD L,A HL points to reversed byte 0.
95CA LD C,(HL) C = reversed byte 0.
95CB INC DE Next: byte 1 index.
95CC LD A,(DE)
95CD LD L,A HL points to reversed byte 1.
95CE LD A,(HL) A = reversed byte 1.
95CF LD (DE),A Write reversed byte 1 in place.
95D0 INC DE Next: byte 2 index.
95D1 LD A,(DE)
95D2 LD L,A HL points to reversed byte 2.
95D3 LD A,(HL) A = reversed byte 2.
95D4 DEC DE Back to byte 0 position.
95D5 DEC DE
95D6 LD (DE),A Write reversed byte 2 to byte 0 position.
95D7 INC DE
95D8 INC DE Advance to byte 2 position.
95D9 LD A,C Recover reversed byte 0 (in C).
95DA LD (DE),A Write it to byte 2 position.
95DB INC DE Advance DE to next scanline.
95DC DJNZ mirror_tile_inplace_1 Repeat for B scanlines.
95DE RET Return.
