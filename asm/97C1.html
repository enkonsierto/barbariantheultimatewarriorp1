97C1: Blit stage background to screen row by row.
Used by the routine at entry_point.
Copies backfround tile graphics and RLE colour attributes from the active background source to VRAM one row at a time, inserting a 112 ms pause after each of the first 15 rows to produce a visible top-to-bottom reveal. The final 2 rows are blitted without a pause. Resets the attribute stream pointer active_bg_attr_ptr to its initial value on exit so the next call starts from the beginning of the attribute table.
INPUT: None. OUTPUT: None. DESTROYS: AF, BC, DE, HL.
paints_stage_screen 97C1 LD HL,($B57D) Load the initial attribute stream address from bg_1_attr_data and preserve it on the stack for the reset at the end.
97C4 PUSH HL
97C5 LD DE,($B57B) Load the active stage graphics source address from active_bg_bitmap_ptr into DE (stage_1_bg stage 1, stage_2_bg stage 2).
97C9 LD HL,$40E2 Set HL to the VRAM address of the first tile of the game window.
97CC LD B,$0F B=15, one iteration per row for the top section; the last 2 rows are handled separately without a pause.
paints_stage_screen_0 97CE PUSH BC Preserve the loop counter.
97CF CALL copy_bg_row Copy one full tile row (28 tiles, 8 scanlines) from source to VRAM via copy_bg_row; DE and HL advance to the next row.
97D2 CALL apply_bg_attr_row Apply RLE colour attributes to the current VRAM row via apply_bg_attr_row; active_bg_attr_ptr advances to the next attribute entry.
97D5 CALL waits_112ms Wait 112 ms via waits_112ms.
97D8 POP BC Restore the loop counter.
97D9 DJNZ paints_stage_screen_0 Repeat for all 15 rows.
97DB LD HL,$57C2 Advance DE and HL to the penultimate row.
97DE CALL apply_bg_attr_row Copy the penultimate row via copy_bg_row.
97E1 CALL waits_112ms Apply attributes to the penultimate row via apply_bg_attr_row.
97E4 LD HL,$57E5 Advance DE and HL to the last row.
97E7 CALL apply_bg_attr_row Copy the last row via copy_bg_row.
97EA CALL waits_112ms Apply attributes to the last row via apply_bg_attr_row.
97ED POP HL Restore initial attribute stream address from the stack.
97EE LD ($B57D),HL Write restored address to active_bg_attr_ptr, resetting the attribute stream pointer for the next call.
97F1 RET Return.
