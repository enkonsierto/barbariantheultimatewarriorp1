B6E1: Player 2 overlay state block.
9-byte block reusing the same layout as the head state block at head_state, allowing render_head_tile and head_mirroring to serve both renderers without modification.
Written each frame by p2_overlay_tile before jumping to render_head_tile.
p2_overlay_block_state B6E1 DEFB $00 IX+0: mirror flag (0=facing right, 1=facing left). Copied from p2_mirror_flags bit 0.
B6E2 DEFB $01 IX+1: not written by p2_overlay_tile; not read.
B6E3 DEFW $B873 IX+2,3: pointer to current tile index entry = p2_overlay_tile_table + P2 frame index.
B6E5 DEFW $0000 IX+4,5: not used by P2 overlay path. Retains head state values.
B6E7 DEFW $5E0A IX+6,7: framebuffer destination address. Computed by p2_overlay_tile.
