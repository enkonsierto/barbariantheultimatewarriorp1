9693: Copy stage background to framebuffer.
Used by the routine at entry_point.
Fills the framebuffer with the active stage background graphics each frame, ready for sprite rendering.
Copies 94 scanlines of 28 bytes each via 28 unrolled LDI instructions per row. After each row, four INC DE instructions skip the 4-byte border gap, keeping the destination aligned to the next framebuffer row (stride 32, graphics width 28).
INPUT: None. OUTPUT: None. DESTROYS AF, BC, DE, HL.

969A LD DE,$5E02 Set DE to framebuffer, framebuffer start.
969D LD B,$5E B = 94, total scanline rows to copy.
969F LD C,$00 C = 0, initialises BC so LDI decrements BC correctly each copy.
