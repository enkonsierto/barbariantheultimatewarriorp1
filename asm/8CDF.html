8CDF: Decimal number renderer.
Converts an unsigned 16-bit value to decimal digits and draws them using the UDG font. Used at paint_score.
Extracts each digit from most to least significant using repeated subtraction against a divider table pointed by IX (e.g. 10000, 1000, 100, 10, 1). Each digit is converted to ASCII (add 48) and drawn via draw_character. Uses the shadow register set to preserve the digit count and value across draw calls.
Callers may set IX before jumping to print_dec_number_0 to use a narrower divider table (e.g. B9A4 for 2-digit display).
INPUT: B = number of digits to print (1-5). DE value to convert. HL = start VRAM address. IX = divider table address.
OUTPUT: None. DESTROYS: AF, BC, DE, HL, IX and shadow registers.
print_dec_number 8CDF LD IX,$B99E Loads the default 5-digit divider table divder_table into IX; callers may skip this instruction by jumping to print_dec_number_0 with a custom IX.
This entry point is used by the routines at entry_point and update_time_display.
print_dec_number_0 8CE3 PUSH BC Preserves B and DE, switches to shadow registers, moves the value into HL and restores the digit count in B.
8CE4 PUSH DE
8CE5 EXX
8CE6 POP HL
8CE7 POP BC
print_dec_number_1 8CE8 LD E,(IX+$00) Loads the current divider into DE and initialises the digit counter C to 255.
8CEB LD D,(IX+$01)
8CEE LD C,$FF Clears carry before the subtraction loop.
8CF0 AND A
print_dec_number_2 8CF1 SBC HL,DE Subtracts the divider from HL repeatedly, counting in C, until underflow; restores the last valid HL.
8CF3 INC C
8CF4 JR NC,print_dec_number_2
8CF6 ADD HL,DE
8CF7 LD A,C Converts the digit count in C to ASCII (add 48) and returns to the main register set.
8CF8 ADD A,$30
8CFA EXX
8CFB PUSH HL Draws the digit via draw_character, then advances HL to the next character column.
8CFC CALL draw_character
8CFF POP HL
8D00 INC HL
8D01 EXX Switches back to the shadow set and advances IX to the next divider entry.
8D02 INC IX
8D04 INC IX
8D06 DJNZ print_dec_number_1 Decrements B and loops for the remaining digits.
8D08 RET Return.
