8C46: Key state tester
Used by the routines at entry_point, pause_handler_routine, human_input_reader, main_menu, one_player_mode and two_player_mode.
Converts the scancode and returns if the key is pressed or not in the carry flag.
INPUT: A the scancode.
OUTPUT: Current key state in the carry flag (1 released, 0 pressed).
DESTROYS: A, BC, carry.
key_state_test 8C46 LD C,A C scancode to test.
8C47 AND $07 Extract row index bits 5-3 add 1 to get 1-based row number in A and B.
8C49 INC A
8C4A LD B,A B row number used as shift counter.
8C4B SRL C Shift row bits into the low position to isolate the row index in A.
8C4D SRL C
8C4F SRL C
8C51 LD A,$05 Extract column index bits 2-0 into A.
8C53 SUB C
8C54 LD C,A C column index bit position within the row byte.
8C55 LD A,$FE Form keyboard port high byte for this row 0xFE base.
key_state_test_0 8C57 RRCA Shift row mask B times to select the correct keyboard half-row port address.
8C58 DJNZ key_state_test_0
8C5A IN A,($FE) Read 5-key state from the ULA keyboard port into A.
key_state_test_1 8C5C RRA Shift right C times to place the target key bit into the carry flag.
8C5D DEC C
8C5E JR NZ,key_state_test_1
8C60 RET Returns with carry set key released, carry clear key pressed.
