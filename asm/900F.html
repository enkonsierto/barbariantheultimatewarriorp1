900F: Calculates the VRAM address of the scanline above.
Given a VRAM scanline address in HL, steps back one scanline, handling tile-row and screen-third boundaries in the non-linear ZX Spectrum screen layout.
INPUT: HL = current VRAM scanline address. OUTPUT: HL = address of the scanline above. DESTROYS: A, F.
calc_prev_scanline_addr 900F DEC H Steps back one scanline (H = $100, 256 bytes).
9010 LD A,H Checks whether the decrement crossed a tile boundary (lower 3 bits of H become 7 when wrapping backward).
9011 CPL
9012 AND $07
9014 RET NZ Within the same tile; address is ready.
9015 LD A,L Moves up one tile row (L encodes the tile row; subtract 32 to reach the row above).
9016 SUB $20
9018 LD L,A
9019 RET C Tile-row underflow carried into the screen-third; address is ready.
901A LD A,H No carry; the screen-third was over-decremented - adds 8 to H to correct it.
901B ADD A,$08
901D LD H,A
901E RET Return.
