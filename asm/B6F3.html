B6F3: P1 head tile sequence.
17-entry tile index sequence for the severed P1 head animation.
Consumed by head_animator one index per frame via the tile sequence pointer at IX+2,3 (head_frame_seq). Each byte is a tile index resolved against sprite bank base other_sprites by render_head_tile.
Terminated by sentinel 255 at B704, which triggers mark_head_landed to mark the head as landed (head_state_flag = 255).
Tile base: other_sprites. 17 active tile IDs + 1 sentinel = 18 bytes total.
p1_head_tile_seq B6F3 DEFB $01,$02,$04,$05 Tile IDs: 1,2,4,5 (cycle 1).
B6F7 DEFB $01,$02,$04,$05 Tile IDs: 1,2,4,5 (cycle 2).
B6FB DEFB $01,$02,$04,$05 Tile IDs: 1,2,4,5 (cycle 3).
B6FF DEFB $01,$01,$02,$02,$02 Tile IDs: 1,1,2,2,2 (final phase).
B704 DEFB $FF Sentinel 255: end of sequence, triggers mark_head_landed.
