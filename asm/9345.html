9345: Activate goblin and initialise its state block.
Called exclusively by goblin_entry once all spawn guard conditions pass.
Sets IY to goblin_state_block (goblin state block base) and calls set_char_moves with movement ID 30 to initialise IY+2,4,5,6,8 and start the goblin entrance animation. Then, writes 255 to goblin_state_block (IY+0) as the active sentinel.
0 = goblin inactive (cleared by combat_loop when body exits screen at X=29).
255 = goblin active (set here, guards re-entry in goblin_entry).
INPUT: A = 30 (movement ID for goblin entrance animation). OUTPUT: None. DESTROYS: AF, BC, IY.
init_goblin 9345 LD IY,$B56B IY = goblin_state_block goblin state block base address.
9349 CALL set_char_moves Call set_char_moves with A=30 to initialise movement and animation data.
934C LD A,$FF A = 255: goblin active sentinel.
934E LD ($B56B),A Write 255 to goblin_state_block: marks goblin as active for goblin_entry and combat_loop guard checks.
9351 RET Return.
