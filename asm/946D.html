946D: Flying head animator.
Called once per frame from the main game loop via 89F7.
Moves and renders the severed head using the 8-byte state block at head_state, addressed via IX.
Movement control data: 2-byte pairs (control byte, magnitude). Bit 0 = move up (move_scanl_up), bit 1 = move down (move_scanl_down). Pointer at IX+4,5 advances 2 bytes per frame.
Tile sequence at IX+2,3 advances 1 byte per frame. When the byte value is 255, INC A wraps to 0 and triggers mark_head_landed: sets head_state_flag=255 (landed), unblocking goblin spawn in goblin_entry.
After horizontal movement, column byte (L AND 31) is checked against 0, 1 and 29. Any edge column triggers deactivate_head: sets head_state_flag=0 (fully inactive).
Landed path: head_state_flag=255, INC A wraps to 0, JR Z jumps to render_head_tile skipping all movement logic. The head is redrawn in place each frame until goblin_kicks re-activates it.
IN None. OUT None. DESTROYS AF, BC, DE, HL, IX.
head_animator 946D LD IX,$B6D9 IX = head_state head state block base.
9471 LD A,(IX+$01) A = head_state_flag state byte.
9474 OR A Test if inactive (state = 0).
9475 RET Z Return if inactive.
9476 INC A INC A: wraps 255 (landed) to 0.
9477 JR Z,render_head_tile Landed: jump to draw-only path at render_head_tile.
9479 CALL shadow_head Bouncing: draw ground shadow via shadow_head.
947C CALL render_head_tile Draw head tile at current position via render_head_tile.
947F LD L,(IX+$04) HL = IX+4,5 (movement control pointer, head_traj_ptr).
9482 LD H,(IX+$05)
9485 LD C,(HL) C = control byte (bit 0=up, bit 1=down).
9486 INC HL Advance past control byte.
9487 LD B,(HL) B = magnitude (scanlines to move this frame).
9488 LD L,(IX+$06) HL = IX+6,7 (current framebuffer address, head_screen_addr).
948B LD H,(IX+$07)
948E BIT 0,C Bit 0 set: move up.
9490 CALL NZ,move_scanl_up Move up B scanlines via move_scanl_up.
9493 BIT 1,C Bit 1 set: move down.
9495 CALL NZ,move_scanl_down Move down B scanlines via move_scanl_down.
9498 LD A,(IX+$00) A = IX+0 direction flag.
949B CP $01 Direction = left?
949D CALL Z,move_col_left Yes: DEC L (move one column left) via move_col_left.
94A0 CALL NZ,move_col_right No: INC L (move one column right) via move_col_right.
94A3 LD (IX+$06),L Store updated framebuffer address back to IX+6,7 (head_screen_addr).
94A6 LD (IX+$07),H
94A9 LD A,L Read column byte from updated framebuffer address (L).
94AA AND $1F Mask to 5 bits to get screen column (0-31).
94AC OR A Column = 0: left edge, deactivate.
94AD JR Z,deactivate_head Deactivate via deactivate_head.
94AF CP $01 Column = 1: near left edge, deactivate.
94B1 JR Z,deactivate_head Deactivate via deactivate_head.
94B3 CP $1D Column = 29: right edge, deactivate.
94B5 JR Z,deactivate_head Deactivate via deactivate_head.
94B7 LD L,(IX+$02) HL = IX+2,3 (tile animation sequence pointer, head_frame_seq).
94BA LD H,(IX+$03)
94BD INC HL Advance sequence pointer by 1.
94BE LD A,(HL) A = current tile index byte.
94BF INC A INC A: test for end-of-sequence sentinel (255 wraps to 0).
94C0 JR Z,mark_head_landed Sentinel: mark head as landed via mark_head_landed.
94C2 LD (IX+$02),L Store updated tile sequence pointer back to IX+2,3 (head_frame_seq).
94C5 LD (IX+$03),H
94C8 LD L,(IX+$04) HL = IX+4,5 (movement control pointer, head_traj_ptr).
94CB LD H,(IX+$05)
94CE INC HL Advance movement control pointer past the consumed pair (2 bytes).
94CF INC HL
94D0 LD (IX+$04),L Store updated movement control pointer back to IX+4,5 (head_traj_ptr).
94D3 LD (IX+$05),H
94D6 RET Return.
