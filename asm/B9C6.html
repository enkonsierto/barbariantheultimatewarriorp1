B9C6: Fight counter.
Tracks how many combats have been played in the current session. Range 0-7.
Read by post_fight_handler and update_time_display.
RESET to 0 by post_fight_handler only when a fight ends by timeout timer_flag=0. On normal wins the counter is not reset here.
initialised to 255 at 89B3. First call to reset_p2_state_block wraps it to 0. Incremented at the start of each fight. When counter reaches 7, post_fight_handler no longer calls R35458, so stage stops alternating from fight 9 onwards. This 8-fight limit is inherited from 1-player mode which has exactly 8 fights.
fight_counter B9C6 DEFB $00 Fight counter. 0-7, capped at 7 from fight 8 onwards.
