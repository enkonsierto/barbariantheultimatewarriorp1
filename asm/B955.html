B955: Sinclair Interface 2 port 2 fixed scancode table.
5-byte read-only table for controller type 3 (Sinclair Interface 2, keys 6-0).
Values: 35,27,19,3,11 (keyboard keys 0,9,8,6,7).
Never written at runtime. Read by human_input_reader when controller type is 3.
sinclair_joystick_2_table B955 DEFB $23 Fire = key 0
B956 DEFB $1B Up = key 9
B957 DEFB $13 Down = key 8
B958 DEFB $03 Left = key 6
B959 DEFB $0B Right = key 7
