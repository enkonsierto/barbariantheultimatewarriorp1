9249: Draws one sprite tile to the framebuffer.
Used by the routine at draws_4_x_3_tiles.
Used by the 4x3 tile renderer to draw individual tiles in a character sprite.
Skips the draw if the tile index is 255 (empty slot).
Applies mirror correction to the tile index if needed based on direction.
INPUT: IX = column offset and sentinel; IY = character state block.
OUTPUT: C adjusted for next tile position (INC/DEC based on mirror flag). DESTROYS: AF, B, DE, HL.
