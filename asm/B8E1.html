B8E1: Per-frame overlay position table.
One byte per P2 animation frame. Encodes the framebuffer offset for the overlay relative to the P2's tile:
bits 0-5 = scanline offset downward from framebuffer row 94 (0-63 lines).
bits 6-7 = fine column offset rightward from P2 X tile (0-3 tiles).
When P2 is mirrored (p2_mirror_flags bit 0 set), p2_overlay_tile inverts the column offset as 6-L before adding to the X tile.
Indexed by P2 frame index p2_frame_index. Read by p2_overlay_tile via get_byte_value.
