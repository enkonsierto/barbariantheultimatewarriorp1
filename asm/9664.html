9664: Interrupt Service Routine
Triggered by Z80 interrupt mode 2 (IM 2).
In execution time, this routine is written by BF87 to 0xF0F0/61680 (more info in F081) so the Vector table can call it.
Two responsibilities: frequency divider via isr_counter, combat timer decrement via timer_flag.
OUTPUT: isr_counter incremented or reset; timer_flag decremented.
int_serv_routine 9664 PUSH AF Preserve AF, BC
9665 PUSH BC
9666 LD A,($B972) isr_counter (ISR frame counter, cycles 0-8).
9669 INC A Advance counter.
966A CP $09 Has the counter reached 9 (one timer tick)?
966C JR Z,int_serv_routine_0 Yes: reset counter and process timer.
966E LD ($B972),A No: store updated counter and exit ISR.
9671 JR int_serv_routine_1 Jump to exit.
int_serv_routine_0 9673 XOR A Counter reached 9 and is reset to 0.
9674 LD ($B972),A
9677 LD A,($B6D8) timer_flag (combat timer: 90->0 in 2P mode; 255 = infinite/1P).
967A CP $FF Is it infinite-time mode (1P/Demo)?
967C JR Z,int_serv_routine_1 Yes: skip decrement, exit.
967E OR A Is timer already 0?
967F JR Z,int_serv_routine_1 Yes: already expired, skip decrement, exit.
9681 LD B,A No: preserve timer value in B.
9682 LD A,($B56B) (goblin active flag: 0=inactive, 255=active). Is the goblin currently active?
9685 CP $FF
9687 LD A,B Restore timer value into A.
9688 JR Z,int_serv_routine_1 Goblin active: freeze timer, exit without decrementing.
968A DEC A Decrement combat timer by 1.
968B LD ($B6D8),A Store updated timer value back to timer_flag.
int_serv_routine_1 968E POP BC Restore BC and AF.
968F POP AF
9690 EI Re-enable interrupts.
9691 RETI Return from interrupt.
