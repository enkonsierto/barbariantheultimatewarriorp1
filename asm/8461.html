8461: Converts note value to period.
Converts a note value to a time period for the beeper.
Adds 12 to the value (transposition) and uses note_table as period table.
INPUT: HL = address of note buffer (melody_note or bass_note).
OUTPUT: H = note period, L = 1, DE = note value + 12 (Octave).
DESTROYS: A.
note_to_period_converter 8461 LD A,(HL) Reads note value from buffer.
8462 ADD A,$0C Adds 12 (octave/transposition adjustment).
8464 LD E,A E = adjusted note value.
8465 LD D,$00 D = 0.
8467 LD HL,$8504 HL = base address of period table note_table.
846A ADD HL,DE HL = table + note index.
846B LD H,(HL) H = period from table.
846C LD L,$01 L = initial half-cycle counter value.
846E RET Returns with period in HL.
