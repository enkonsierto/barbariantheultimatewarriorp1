8A56: Sets infinite time or not.
Sets the timer depending if there are two or one (human) players.
Called on each game restart via 89D8. If timer_flag is already 255 (initial value on load, or 1P/demo mode), returns immediately.
Otherwise writes 101 to timer_flag for 2P mode: the extra counts above 90 compensate for the timer decrement between menu selection and game start, so the HUD reads 90 when combat begins.
Subsequent rounds bypass this routine and load 90 directly via 8EA7.
set_infinite_time 8A56 LD A,($B6D8) Load round timer timer_flag into A.
8A59 CP $FF If 255, already in infinite time mode; return.
8A5B RET Z Return.
8A5C LD A,$65 A = 101: 2 Player mode.
8A5E LD ($B6D8),A Later 90 will be written in the timer_flag via 8EA7.
8A61 RET Return.
