90FA: Generate right-edge column mask for partial tile blending.
Called from right_edge_renderer for non-0xFF tile bytes at right tile boundary.
Counts trailing 1-bits from bit 0 (LSB) to find first transparent column, loads LSB-packed mask (1,3,7,15,31,63,127) into A and jumps to mask_and_blend.
INPUT: A = right-edge tile byte, HL = framebuffer address, DE = tile source.
OUTPUT: A = pixel mask or 0 (fully transparent). DESTROYS: A, C.
right_mask_gen 90FA LD C,A C = right-edge tile byte. Tests bit 0: set means at least the rightmost pixel column is visible.
90FB BIT 0,C Bit 0 clear: byte is 0x00, no visible columns - jump to mask_byte_clear.
90FD JR Z,mask_byte_clear
90FF LD A,$01 Bit 0 set. A = 0x01 (00000001b); test bit 1.
9101 BIT 1,C Bit 1 clear: 1 visible column - jump to mask_and_blend with mask 0x01.
9103 JR Z,mask_and_blend
9105 LD A,$03 A = 0x03 (00000011b); test bit 2.
9107 BIT 2,C Bit 2 clear: 2 visible columns - jump to mask_and_blend with mask 0x03.
9109 JR Z,mask_and_blend
910B LD A,$07 A = 0x07 (00000111b); test bit 3.
910D BIT 3,C Bit 3 clear: 3 visible columns - jump to mask_and_blend with mask 0x07.
910F JR Z,mask_and_blend
9111 LD A,$0F A = 0x0F (00001111b); test bit 4.
9113 BIT 4,C Bit 4 clear: 4 visible columns - jump to mask_and_blend with mask 0x0F.
9115 JR Z,mask_and_blend
9117 LD A,$1F A = 0x1F (00011111b); test bit 5.
9119 BIT 5,C Bit 5 clear: 5 visible columns - jump to mask_and_blend with mask 0x1F.
911B JR Z,mask_and_blend
911D LD A,$3F A = 0x3F (00111111b); test bit 6.
911F BIT 6,C Bit 6 clear: 6 visible columns - jump to mask_and_blend with mask 0x3F.
9121 JR Z,mask_and_blend
9123 LD A,$7F A = 0x7F (01111111b); bits 6..0 all set: 7 visible columns.
9125 JR mask_and_blend Jump to mask_and_blend with mask 0x7F.
