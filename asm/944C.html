944C: Goblin kicks the severed head off-screen.
Used by the routine at combat_loop.
Called from combat_loop when the goblin X position matches the head screen column.
Rewinds the tile sequence pointer by 16: reads the current pointer from head_frame_seq, subtracts 16 to recover the original sequence base (p1_head_tile_seq P1 or p2_head_tile_seq P2), and passes it to head_reactivate as BC. This restarts the head tile animation from frame 0 for the kick arc.
Switches the trajectory to kicked_traj_data (kicked-head bounce), which opens with a sharp upward arc (6 scanlines up per frame, control byte bit 0 = move-up).
INPUT: B = Goblin X position. OUTPUT None. DESTROYS: AF, DE, HL.
