8BB5: Character AI engine.
Called by input_ai_dispatcher with A = extra state byte (p2_frame_index for P1, p1_frame_index for P2) and IY pointing to the active character state block.
Performs a 4-step lookup chain:
1. extra state -> AI profile block (ai_profile_0) via pointer table ai_profile_table.
2. Fighter distance (fighter_distance) -> profile column 0-7 via second_ai_state_table.
3. Profile column -> movement block ID (0-22) -> 8-byte row in ai_movement_table (x8 offset).
4. R register (pseudo-random, masked 0-31) + fight counter (fight_counter) -> difficulty entry (ai_difficulty_table) -> subindex (0-7) -> movement ID from ai_movement_table.
Maps the movement ID to a key-state bitmask and stores it in IY+1 so the movement engine sees it as real input.
IN A = extra state byte. IY = character state block.
OUT (IY+1) = AI-generated key state bitmask.
DESTROYS AF, BC, DE, HL.
character_ai_engine 8BB5 LD HL,$BA02 HL = profile pointer table base ai_profile_table. A = extra state index.
8BB8 CALL solve_pointer Resolve AI profile pointer via solve_pointer; HL = profile block address in ai_profile_0.
8BBB PUSH HL Preserve profile block pointer.
8BBC LD A,($B6F2) A = fighter distance fighter_distance.
8BBF LD HL,$B9DA HL = secondary AI state-to-column table second_ai_state_table.
8BC2 CALL get_byte_value Map distance to profile column (0-7) via get_byte_value; A = column index.
8BC5 POP HL Restore profile block pointer.
8BC6 CALL get_byte_value Read movement block ID from profile at column A via get_byte_value; A = block ID (0-22).
8BC9 LD DE,$0008 DE = 8 (byte stride per block in ai_movement_table).
8BCC CALL bin_multiply HL = A * 8 via bin_multiply; byte offset into the movement selection table.
8BCF LD DE,$BB66 DE = movement table base ai_movement_table.
8BD2 ADD HL,DE HL = table base + offset; points to the selected 8-byte movement block.
8BD3 PUSH HL Preserve movement block pointer.
8BD4 LD A,R A = R register; pseudo-random source.
8BD6 AND $1F Mask A to 0-31; random index into the difficulty entry.
8BD8 PUSH AF Preserve random value.
8BD9 LD A,($B9C6) A = fight counter fight_counter (0-7); difficulty level index.
8BDC LD HL,$B9C7 HL = difficulty table base ai_difficulty_table.
8BDF CALL solve_pointer Resolve difficulty entry pointer for this fight level via solve_pointer.
8BE2 POP AF Restore random value.
8BE3 CALL get_byte_value Index into difficulty entry using random value via get_byte_value; A = subindex (0-7).
8BE6 POP HL Restore movement block pointer.
8BE7 CALL get_byte_value Read movement ID from block at subindex A via get_byte_value; A = movement ID.
8BEA CALL input_mirror_correcter Mirror movement ID if necessary, via input_mirror_correcter (input-class table input_lookup_table, mirror-corrected for IY+2 bit 0).
8BED LD (IY+$01),A Store AI key bitmask in (IY+1).
8BF0 RET Return.
