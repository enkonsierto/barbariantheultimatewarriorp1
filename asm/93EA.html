93EA: Decapitation setup.
Called from sprite_dispatcher when the sprite dispatcher encounters frame index 17.
IY points to the attacker state block on entry.
Identifies the defender (opposite character) and applies movement 37 (decapitation fall) to them via set_char_moves. Then, if no head is already in flight (head_state_flag != 1), computes the head spawn position in the framebuffer from the attacker X tile column plus a 15-row vertical offset, and initialises the flying-head state block at head_state via head_init.
IN: IY attacker state block (p1_controller_type or p2_controller_type).
OUT: None. head_state initialised if head_state_flag != 1.
DESTROYS AF, AF', IY.
decap_setup 93EA PUSH AF Preserve AF (frame index that triggered this call).
93EB PUSH IY Preserve IY (attacker state block).
93ED LD A,IYl A = attacker type: low byte of IY base address (89=P1, 98=P2).
93EF LD IY,$B559 Default: IY = P1 state block R46425 (defender if attacker is P2).
93F3 CP $59 Is attacker P1 (type 89)?
93F5 JR NZ,decap_setup_0 No (attacker is P2) -> defender is P1, keep IY = p1_controller_type.
93F7 LD IY,$B562 Yes (attacker is P1) -> defender is P2, set IY = p2_controller_type.
decap_setup_0 93FB LD A,$25 A = 37: movement code for decapitation fall.
93FD CALL set_char_moves Apply movement 37 to the defender via set_char_moves.
9400 POP IY Restore IY: now points to the attacker block.
9402 LD A,($B6DA) Read head active state from head_state_flag.
9405 CP $01 Is head already in flight (state = 1)?
9407 JR Z,decap_setup_3 Yes -> skip initialisation, jump to decap_setup_3.
9409 LD A,(IY+$03) A = attacker X tile position (IY+3).
940C LD L,A L = column offset in framebuffer.
940D LD H,$5E H = 0x5E: framebuffer base framebuffer.
940F LD DE,$01E0 DE = 480 (15 rows * 32): vertical offset to head spawn row.
9412 ADD HL,DE HL = framebuffer address at attacker column, 15 rows down.
9413 LD A,(IY+$02) A = mirror flags byte (IY+2).
9416 AND $01 Isolate bit 0: mirror flag (1 = attacker facing left).
9418 JR Z,decap_setup_1 Not mirrored -> head spawns directly at attacker column, no adjustment needed.
941A LD DE,$0006 DE = 6: hardcoded horizontal offset for head spawn position (mirrored sprites only).
941D ADD HL,DE Shift head spawn point 6 bytes right.
decap_setup_1 941E EX AF,AF' Save mirror bit to AF', free A for next operations.
941F XOR A A = 0.
9420 LD ($B6E9),A Write 0 to unused_buffer (redundant: always 0, never read from this buffer).
9423 LD DE,$B7A9 DE = normal_traj_data: 18-entry head flight trajectory table (Dx,Dy pairs per frame).
9426 LD BC,$B705 BC = p2_head_tile_seq: P2 head tile sequence (default, used when attacker is P2).
9429 LD A,IYl A = attacker type (IY+0 low byte).
942B CP $62 Is attacker P2 (type 98)?
942D JR Z,decap_setup_2 Yes -> keep BC = p2_head_tile_seq (P2 head tiles), jump to decap_setup_2.
942F LD BC,$B6F3 No (attacker is P1) -> BC = R46835 (P1 head tiles).
decap_setup_2 9432 EX AF,AF' Restore A = mirror bit from AF'.
9433 CALL head_init Initialise flying-head state block head_state via head_init.
decap_setup_3 9436 POP AF Restore AF.
9437 RET Return.
