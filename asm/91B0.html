91B0: Edge-mask sub-tile renderer.
Used by the routine at comp_tile_dispatcher.
Renders edge-masked sub-tile (left/right via bit6); extracts scanlines to B (bits4-0).
Bit6=0: left-edge AND-mask via tile_render_engine (entry 36943). Bit6=1: full AND-mask via right_edge_loop_0.
Advances BC past current byte; returns next definition byte in A (bit5=1 ends sequence).
IN: A=definition byte (bits7-6=01/10, bit6=renderer, bits4-0=scanlines). BC=def pointer.
OUT: A=next def byte. BC=advanced. DESTROYS: AF, B, BC.
edge_mask_renderer 91B0 PUSH BC Saves def pointer (BC); C=def byte (bit6 tested after B set).
91B1 LD C,A
91B2 AND $1F Extracts scanline count to B (bits4-0).
91B4 LD B,A
91B5 BIT 6,C Tests bit6 of C: 0=left-edge AND-mask, 1=AND-mask loop.
91B7 JR NZ,edge_mask_renderer_0 Bit6=1: full AND-mask via right_edge_loop_0 (B scanlines).
91B9 CALL left_edge_loop Bit6=0: left-edge AND-mask via tile_render_engine (entry left_edge_loop, B scanlines).
91BC JR edge_mask_renderer_1 Jump to common exit.
edge_mask_renderer_0 91BE CALL right_edge_loop_0 AND-mask via right_edge_loop_0
edge_mask_renderer_1 91C1 POP BC Restores def pointer; advances BC to next def byte.
91C2 INC BC
91C3 LD A,(BC) Loads next def byte to A; returns for end-test in caller.
91C4 RET Return: A = next byte for end-of-def test in comp_tile_dispatcher.
