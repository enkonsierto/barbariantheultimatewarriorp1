920B: Rectangular screen copy.
Used by the routine at draw_snakes_sprites.
Copies B bytes x C scanlines from DE to screen HL.
Advances scanline via calc_next_scanline_addr after each row.
Used for snake HUD (draw_snakes_sprites), row clears (sweep_transition) and stage blits.
INPUT: B = bytes per row, C = row count, DE = source, HL = framebuffer destination.
OUTPUT: None. DESTROYS: AF,BC,DE,HL.
copy_rect 920B PUSH BC Saves row counter.
920C PUSH HL Saves start-of-row destination.
copy_rect_0 920D LD A,(DE) Copies one byte from source to destination.
920E LD (HL),A
920F INC HL Advances source and destination pointers.
9210 INC DE
9211 DJNZ copy_rect_0 Repeats for all bytes in row.
9213 POP HL Restores start-of-row destination.
9214 CALL calc_next_scanline_addr Advances to next framebuffer scanline via calc_next_scanline_addr.
9217 POP BC Restores row counter.
9218 DEC C Decrements row counter.
9219 JR NZ,copy_rect Repeats for all rows.
921B RET Returns.
