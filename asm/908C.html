908C: Right-edge tile scanline renderer.
Renders one scanline of a 3-byte-wide tile onto the framebuffer, scanning from right to left.
Locates the rightmost opaque byte (value != 255), applies the right-edge column mask via right_mask_gen, then direct-blits any remaining bytes to its left.
Counterpart to left_edge_renderer, which masks the left edge via left_mask_gen.
Transparent sentinel is 255. Tile bytes are ordered left to right: byte 0 (leftmost), byte 1 (middle), byte 2 (rightmost).
IN: HL framebuffer scanline start address. DE tile source row start address.
OUT: HL advanced by 32 to the next scanline. DE advanced by 3 to the next tile row.
DESTROYS AF, C.
right_edge_renderer 908C LD C,$FF Set transparent sentinel (255) in C.
908E INC L Advance both pointers to tile byte 2 (rightmost).
908F INC L
