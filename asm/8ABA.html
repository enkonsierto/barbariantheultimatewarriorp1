8ABA: Pause handler.
Called from pause_handler_entry.
Entered from the main game loop when the pause key pause_scancode is pressed.
Plays a sound via sound_head_cut, waits 112ms to debounce the initial press, then polls pause_scancode via key_state_test until the pause key is pressed again to resume.
IN None. OUT None. DESTROYS AF, BC.
pause_handler_routine 8ABA CALL sound_head_cut Play pause sound via sound_head_cut.
8ABD CALL waits_112ms Wait 112ms to absorb the initial key press.
pause_handler_routine_0 8AC0 LD A,($B94E) Load pause key scancode from pause_scancode.
8AC3 CALL key_state_test Test pause key state via key_state_test. Carry=1 released, carry=0 pressed.
8AC6 JR C,pause_handler_routine_0 Key released - loop back, waiting for the second press.
8AC8 JP game_loop Key pressed again - resume, jump to game loop at game_loop.
