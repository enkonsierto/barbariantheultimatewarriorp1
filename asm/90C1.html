90C1: Generate left-edge column mask for partial tile blending.
Called from left_edge_renderer for non-0xFF tile bytes at left tile boundary.
Counts leading 1-bits from bit 7 (MSB) to find first transparent column, loads MSB-packed mask (128,192,224,240,248,252,254) into A and falls through to mask_and_blend.
INPUT: A = left-edge tile byte, HL = framebuffer address, DE = tile source.
OUTPUT: A = pixel mask or 0 (fully transparent). DESTROYS: A, C.
left_mask_gen 90C1 LD C,A C = A (left-edge tile byte).
90C2 BIT 7,C BIT 7,C: if bit 7 clear, byte is #00 fully transparent - jump mask_byte_clear.
90C4 JR Z,mask_byte_clear
90C6 LD A,$80 A = 0x80 (10000000b) test bit 6.
90C8 BIT 6,C if bit 6 clear: 1 visible column - jump mask_and_blend with mask 0x80.
90CA JR Z,mask_and_blend
90CC LD A,$C0 A = 0xC0 (11000000b) test bit 5.
90CE BIT 5,C if bit 5 clear: 2 visible columns - jump mask_and_blend with mask 0xC0.
90D0 JR Z,mask_and_blend
90D2 LD A,$E0 A = 0xE0 (11100000b) test bit 4.
90D4 BIT 4,C if bit 4 clear: 3 visible columns - jump mask_and_blend with mask 0xE0.
90D6 JR Z,mask_and_blend
90D8 LD A,$F0 A = 0xF0 (11110000b) test bit 3.
90DA BIT 3,C if bit 3 clear: 4 visible columns - jump mask_and_blend with mask 0xF0.
90DC JR Z,mask_and_blend
90DE LD A,$F8 A = 0xF8 (11111000b) test bit 2.
90E0 BIT 2,C if bit 2 clear: 5 visible columns - jump mask_and_blend with mask 0xF8.
90E2 JR Z,mask_and_blend
90E4 LD A,$FC A = 0xFC (11111100b) test bit 1.
90E6 BIT 1,C if bit 1 clear: 6 visible columns - jump mask_and_blend with mask 0xFC.
90E8 JR Z,mask_and_blend
90EA LD A,$FE all bits 7..1 set: 7 visible columns, A = 0xFE (11111110b) fall through to mask_and_blend.
