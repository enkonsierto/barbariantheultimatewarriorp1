AA00: Barbarian sprite shape control table.
21 entries (0-20), 1 byte each. Defines shape/behaviour for each frame's sprite.
Accessed as (IX+0) by sprites_mirror_checker (12-tile mirror checker). Controls tile grid expansion:
255 = empty slot: INC IX (skip), no tile consumed from HL frame data. The rest IDs are active shape: consume 1 tile index from HL, render via draw_single_tile.
Produces variable-width sprites: 0-3 tiles wide (4 rows tall), column offset follows.
