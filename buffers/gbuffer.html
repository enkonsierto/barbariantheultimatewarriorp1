Game status buffer
Address Length Description
5E02 2046
Framebuffer
This whole area will be filled with the framebuffer (24066-26751). Overwritten by copy_bg_to_frmbuff.
6758 296
Framebuffer (cont.)
8448 12
Music engine state buffers.
12-byte working area for the 2-channel beeper music engine. Used by music_engine.
B559 9
Player 1 state block.
9-byte structure used by all combat routines.
B562 9
Player 2 state block.
9-byte structure used by all combat routines.
B56B 9
Goblin state block.
B574 2
Saved stack pointer.
2-byte LE copy of SP redirecting SP to the VRAM address table left_col_addr_table.
B576 1
Active Background index.
0=Background 1 (graphics stage_1_bg, attrs bg_1_attr_data), 1=Background 2 (graphics stage_2_bg, attrs bg_2_attr_data).
B6D3 1
Pending combat sound index.
Stores the index of the combat sound triggered this frame.
B6D4 2
Attacker state block pointer.
2-byte pointer to the attacker's character state block.
B6D6 2
Hit indicator screen address.
2-byte VRAM address where the hit indicator glyph will be drawn.
B6D8 1
Timer Flag.
In this address the time is stored (in 2 players mode). If the value is 255 we are in 1 Player or Demo mode (and we got infinite time).
B6D9 8
Flying head state block.
Initialised by head_init on decapitation (head_state + head_state_flag + head_frame_seq + head_traj_ptr + head_screen_addr).
B6E1 8
Player 2 overlay state block.
9-byte block reusing the same layout as the head state block at head_state, allowing render_head_tile and head_mirroring to serve both renderers without modification.
B6F2 1
Horizontal distance between fighters.
Computed by calc_distance as absolute difference of tile X positions.
B945 2
Player 1 score buffer.
In this address the player's 1 score is stored.
B947 2
Player 2 score buffer.
In this address the player's 2 score is stored.
B949 5
P1 keyboard scancode table.
5-byte table: Fire, Up, Down, Left, Right scancodes for P1 (controller type 1).
B94E 1
Pause key Scancode Buffer
Pause Scancode Buffer only configurable when choosing keyboard.
B94F 1
Quit Scancode Buffer
Quit scancode buffer only configurable when choosing keyboard.
B950 5
P2 keyboard scancode table.
5-byte table: Fire, Up, Down, Left, Right scancodes for P2 (controller type 2).
B95F 3
Mask scratch buffer
Temporary 3-byte copy of framebuffer bytes before AND-masking for sprite transparency. Written by tile_renderer. Read by 9150.
B972 1
ISR frame counter
Incremented each interrupt (50 Hz PAL). When it reaches 9 it is reset to 0.
B973 1
2-frame animation and hit-detection gate counter.
Incremented each frame by frame_counter_hit_inc. Cycles 0-1-0-1.
B9C6 1
Fight counter.
Tracks how many combats have been played in the current session. Range 0-7.
