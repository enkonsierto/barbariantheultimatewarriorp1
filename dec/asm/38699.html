38699: Game window row-by-row wipe to black.
Used by the routines at main_menu and menu_fade_out.
Clears the game window from top to bottom one tile row per 112 ms step, producing a visible wipe effect. Rows that share columns with the snake HUD are cleared via clear_snake_row, which preserves those columns; all other rows use clear_row, which clears all 28 tiles. The two bottom rows at attribute-map level are cleared separately via clear_attr_row.
INPUT: None. OUTPUT: None. DESTROYS A, BC, HL.
sweep_transition 38699 LD HL,16610 Set HL to the first tile of the top game window row.
38702 LD B,2 Set loop counter to 2.
sweep_transition_0 38704 PUSH BC Preserve loop counter before call.
38705 CALL clear_snake_row Clear one snake-edge row preserving HUD columns via clear_snake_row.
38708 POP BC Restore loop counter.
38709 DJNZ sweep_transition_0 Repeat for the top 2 snake-edge rows.
38711 LD B,2 Set loop counter to 2.
sweep_transition_1 38713 PUSH BC
38714 CALL clear_row Clear one full 28-tile row and wait 112 ms via clear_row.
38717 POP BC
38718 DJNZ sweep_transition_1 Repeat for the next 2 full rows.
38720 LD B,3 Set loop counter to 3.
sweep_transition_2 38722 PUSH BC
38723 CALL clear_snake_row Clear one snake-edge row via clear_snake_row.
38726 POP BC
38727 DJNZ sweep_transition_2 Repeat for the next 3 snake-edge rows.
38729 LD B,8 Set loop counter to 8.
sweep_transition_3 38731 PUSH BC
38732 CALL clear_row Clear one full row and wait 112 ms via clear_row.
38735 POP BC
38736 DJNZ sweep_transition_3 Repeat for 8 full rows.
38738 XOR A A=0, black attribute value.
38739 LD HL,23234 Set HL to the penultimate attribute row of the game window.
38742 LD BC,7169 B=28 (full row width), C=1 (single row).
38745 CALL clear_attr_row Clear the penultimate attribute row to black via clear_attr_row.
38748 CALL waits_112ms Wait 112 ms via waits_112ms.
38751 XOR A A=0, black attribute value.
38752 LD HL,23269 Set HL to the last attribute row of the game window.
38755 LD BC,5633 B=22 (erasable tiles in the last partial row), C=1.
38758 CALL clear_attr_row Clear the last row attributes to black via clear_attr_row.
38761 JP waits_112ms Wait 112 ms and return; tail-calls waits_112ms.
