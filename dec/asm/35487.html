35487: Post-fight handler. Stage alternation and counter/time reset.
Used by the routines at entry_point and p2_death_score_update.
Called from 35376 on 2P timeout (timer_flag=0) or from 37796 on 1P win (timer_flag always 255 during combat). Alternates stage via change_stage_ptr unless fight counter has reached 7. Resets fight counter to 0 and restores time to 90 only on timeout (2P mode). On 1P win (time=255), counter advances normally with no reset.
post_fight_handler 35487 LD A,(47558) Read the fight counter buffer fight_counter.
35490 INC A Increment only for comparison - will not be stored.
35491 CP 8 Would counter+1 reach 8? (1P fight limit)
35493 CALL NZ,change_stage_ptr If not fight 8, alternate the stage via change_stage_ptr.
35496 LD A,(46808) Load time buffer timer_flag: 255=won by combat, 0=timeout.
35499 OR A If 0: round ended by timeout (2P mode); if 255: P1 victory by combat.
35500 JR NZ,post_fight_handler_0 If non-zero (P1 victory), skip the time reset.
35502 XOR A Timeout path: Reset the fight counter to 0 at fight_counter.
35503 LD (47558),A
35506 LD A,90 Restore the time to 90 at timer_flag.
35508 LD (46808),A
post_fight_handler_0 35511 JP entry_point_1 Return to the game loop.
