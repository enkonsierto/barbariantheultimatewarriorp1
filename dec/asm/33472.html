33472: Snake sprite data.
Four animation frames (1-byte header + 30 scanlines x 3 bytes each) used by update_snake_frame_id to animate the snake HUD when a fighter takes a hit. energy_dec sets the draw phase index in IY+7 bits 7-4 to 1 on impact; update_snake_frame_id advances
it each frame through the 14-step ping-pong cycle (phases 0-13) via the pointer table at snake_pointer_table, resetting to idle (0) when complete.
P1 snake (VRAM 16416) uses a horizontally mirrored copy via mirror_tile_inplace_0 + mirror_tile_inplace.
P2 snake (VRAM 16445) uses the sprite in its natural orientation.
