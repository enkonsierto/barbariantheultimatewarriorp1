44501: Frame sequences (animation data)
One entry per movement ID. Each sequence is a list of frame indices into barb_ptr_table Barbarian sprite pointer table, terminated by a sentinel byte.
Note: Drax sprites and animations are present in the data but are never triggered in the 2 Players version of the game.
Terminators:
253 - LOOP: on the last frame, jump back to the first frame of the sequence.
254 - HOLD: freeze animation advance; the game logic determines when to proceed to the next frame.
255 - CHAIN: end this sequence and chain into another; the following byte specifies the target movement ID.
