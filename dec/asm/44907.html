44907: Movement transition table.
12 rows x 9 bytes each. Each row is indexed by input class 0-8 via input_lookup_table, and holds the movement ID to transition into for that input.
A value of 255 means no transition is defined for that input combination.
Pointed to by no_fire_transition_table no-fire and fire_transition_table fire pressed.
Multiple movements may share the same row identical transition profile.
Input class columns 0neutral, 1fwd, 2back, 3crouch, 4jump, 5crouch-fwd, 6jump-fwd, 7crouch-back, 8jump-back.
Resolved by moves_solver once per frame per character.
