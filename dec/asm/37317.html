37317: Composite tile pointer resolver.
Converts a composite tile index A (0-63) into a pointer BC to its definition byte sequence in 48269, via the 2-byte pointer table at composite_tile_pointers.
Gets the word offset into composite_tile_pointers, reads the little-endian pointer at that entry and returns it in BC.
INPUT A composite tile index (0-63, derived from control byte AND 0x3F). HL framebuffer address (preserved unchanged). OUTPUT BC pointer to definition bytes in 48269. DESTROYS AF, BC.
comptile_ptr_resolver 37317 PUSH HL Saves framebuffer address.
37318 ADD A,A Computes 2-byte offset into pointer table.
37319 LD C,A Forms full offset from low byte.
37320 LD B,0
37322 LD HL,48433
37325 ADD HL,BC Locates table (composite_tile_pointers) entry for this index.
37326 LD C,(HL) Reads pointer into BC.
37327 INC HL
37328 LD B,(HL)
37329 POP HL Restores framebuffer address.
37330 RET Returns with pointer in BC.
