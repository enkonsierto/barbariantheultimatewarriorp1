46443: Goblin state block.
IY points here in init_goblin. set_char_moves writes IY+2, IY+4, IY+5, IY+6, IY+8.
IY+0: 0=goblin inactive (cleared by combat_loop when X=29), 255=active (set by init_goblin).
goblin_state_block 46443 DEFB 0 IY+0: Active flag (0=inactive, 255=active).
46444 DEFB 0 IY+1: Input state byte. Unused for goblin.
goblin_mirror_flags 46445 DEFB 0 IY+2: Mirror and control flags (bit 0=facing direction, bit 7=non-controllable).
goblin_x_pos 46446 DEFB 0 IY+3: X tile position. Also used as frame counter by combat_loop.
goblin_frame_ptr 46447 DEFW 44522 IY+4+5: Frame data pointer.
goblin_movement_id 46449 DEFB 1 IY+6: Current movement ID (30=goblin entrance).
goblin_energy_byte 46450 DEFB 0 IY+7: Energy byte. Unused for goblin.
goblin_frame_index 46451 DEFB 0 IY+8: Current frame index byte.
