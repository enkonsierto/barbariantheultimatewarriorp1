39846: 112 ms delay loop.
Used by the routines at pause_handler_routine, catch_key, key_assign, menu_fade_out, rainbow_animation, post_drain_wait, sweep_transition, clear_row, clear_snake_row and paints_stage_screen.
Busy-wait timing routine used by menu and animation code. Runs a nested loop over B and C to burn a fixed number of T-states, roughly one frame at 50 Hz.
waits_112ms 39846 LD BC,24672 Load BC with the loop counters (B=96, C96).
waits_112ms_0 39849 DEC B Inner loop: decrement B.
39850 JR NZ,waits_112ms_0 If B is not zero, repeat the inner loop.
39852 DEC C Outer loop: decrement C.
39853 JR NZ,waits_112ms_0 If C is not zero, restart the inner loop.
39855 RET Return after both counters have reached zero (after 9216 loops).
