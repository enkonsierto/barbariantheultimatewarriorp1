38220: Player 2 overlay tile renderer.
Composites a single extra tile on top of the P2 sprite each frame to visually differentiate P2 from P1, which share the same base sprite bank.
p2_overlay_tile_table holds a 1-byte-per-frame overlay tile index (0-52 = tile, 255 = no draw) choosen from (other_sprites)
p2_overlay_pos_table holds a 1-byte-per-frame position table: bits 0-5 = scanline offset down from row 94; bits 6-7 (after 2x RLCA) = column offset 0-3 right of P2 X tile, inverted as 6-L when P2 is mirrored.
The overlay state block at p2_overlay_block_state mirrors the layout of the head state block at head_state, letting render_head_tile and head_mirroring serve both.
INPUT: None. OUTPUT: None. DESTROYS: AF, B, DE, HL, IX.
