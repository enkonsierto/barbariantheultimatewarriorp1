38796: Clear all 28 tiles of one game-window row to black.
Used by the routine at sweep_transition.
Calls clear_scanline_ink_attr once per scanline (8 times, one per pixel row within the tile) to zero the pixel data and set attributes to black. After the loop, HL naturally points to the first scanline of the tile row below. Tail-calls waits_112ms to insert a 112 ms pause, producing the visible row-by-row wipe delay in sweep_transition.
INPUT: HL = VRAM address of the first scanline of the tile row to clear.
OUTPUT: HL = VRAM address of the first scanline of the next tile row.
DESTROYS: A, B, DE.
clear_row 38796 LD B,8 B=8, one iteration per scanline within the tile row.
clear_row_0 38798 PUSH BC Preserve the loop counter.
38799 CALL clear_scanline_ink_attr Clear pixel data and attributes for this scanline via clear_scanline_ink_attr.
38802 CALL calc_next_scanline_addr Advance HL to the next scanline via calc_next_scanline_addr.
38805 POP BC Restore the loop counter.
38806 DJNZ clear_row_0 Repeat for all 8 scanlines of the tile row.
38808 JP waits_112ms Tail-call waits_112ms to wait 112 ms before returning.
