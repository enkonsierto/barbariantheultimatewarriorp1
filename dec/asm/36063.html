36063: Decimal number renderer.
Converts an unsigned 16-bit value to decimal digits and draws them using the UDG font. Used at paint_score.
Extracts each digit from most to least significant using repeated subtraction against a divider table pointed by IX (e.g. 10000, 1000, 100, 10, 1). Each digit is converted to ASCII (add 48) and drawn via draw_character. Uses the shadow register set to preserve the digit count and value across draw calls.
Callers may set IX before jumping to print_dec_number_0 to use a narrower divider table (e.g. 47524 for 2-digit display).
INPUT: B = number of digits to print (1-5). DE value to convert. HL = start VRAM address. IX = divider table address.
OUTPUT: None. DESTROYS: AF, BC, DE, HL, IX and shadow registers.
print_dec_number 36063 LD IX,47518 Loads the default 5-digit divider table divder_table into IX; callers may skip this instruction by jumping to print_dec_number_0 with a custom IX.
This entry point is used by the routines at entry_point and update_time_display.
