33864: Music engine state buffers.
12-byte working area for the 2-channel beeper music engine. Used by music_engine.
The loop-pointer slots at melody_loop_ptr and bass_loop_ptr are unused pointers where play_music would store a fixed restart address. Superseded by restart_music, which reads the restart address inline from the music data stream.
melody_note 33864 DEFB 0 Current melody note index; used by note_to_period_converter to look up period in note_table
bass_note 33865 DEFB 0 Current bass note index; used by note_to_period_converter
beeper_state 33866 DEFB 0 Speaker output bit state, 0 or 16; toggled each half-cycle by music_engine
melody_ptr 33867 DEFW 60000 Current read position in melody sequence melody_data; updated by select_music_pointer
melody_loop_ptr 33869 DEFW melody_data_0 - UNUSED. Melody loop-back address; never written in the final build
bass_ptr 33871 DEFW 61000 Current read position in bass sequence bass_data; updated by select_music_pointer
bass_loop_ptr 33873 DEFW 34672 - UNUSED. Bass loop-back address; never written in the final build
note_duration 33875 DEFB 240 Duration counter for the current note; loaded each note by music_engine
