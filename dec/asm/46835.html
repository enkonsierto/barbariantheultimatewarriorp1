46835: P1 head tile sequence.
17-entry tile index sequence for the severed P1 head animation.
Consumed by head_animator one index per frame via the tile sequence pointer at IX+2,3 (head_frame_seq). Each byte is a tile index resolved against sprite bank base other_sprites by render_head_tile.
Terminated by sentinel 255 at 46852, which triggers mark_head_landed to mark the head as landed (head_state_flag = 255).
Tile base: other_sprites. 17 active tile IDs + 1 sentinel = 18 bytes total.
p1_head_tile_seq 46835 DEFB 1,2,4,5 Tile IDs: 1,2,4,5 (cycle 1).
46839 DEFB 1,2,4,5 Tile IDs: 1,2,4,5 (cycle 2).
46843 DEFB 1,2,4,5 Tile IDs: 1,2,4,5 (cycle 3).
46847 DEFB 1,1,2,2,2 Tile IDs: 1,1,2,2,2 (final phase).
46852 DEFB 255 Sentinel 255: end of sequence, triggers mark_head_landed.
