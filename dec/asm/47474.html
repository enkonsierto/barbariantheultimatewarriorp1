47474: ISR frame counter
Incremented each interrupt (50 Hz PAL). When it reaches 9 it is reset to 0.
Written and read exclusively by int_serv_routine (ISR).
Acts as a frequency divider: the combat timer timer_flag is decremented once every 9 interrupts, giving an effective timer rate of 50/9 ~= 5.56 Hz.
Initialised to 0 by entry_point_1 at the start of each game round.
isr_counter 47474 DEFB 0 ISR frame counter. Range 0-8. Resets to 0 on tick 9.
