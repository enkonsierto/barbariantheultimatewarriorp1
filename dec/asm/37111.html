37111: Write transparent tile byte to framebuffer.
Shared exit for left_mask_gen/right_mask_gen when tile byte has no visible columns on edge.
Tile encoding guarantees bit 7 (left) or bit 0 (right) clear = entire byte 0x00. Writes 0 to clear framebuffer destination byte, preserving background.
INPUT: C = tile byte (0x00), HL = framebuffer address.
OUTPUT: None. DESTROYS: A.
mask_byte_clear 37111 LD A,C A = C (0x00 - no sprite pixels on this edge); Write 0 to framebuffer byte at HL.
37112 LD (HL),A
37113 RET Return.
