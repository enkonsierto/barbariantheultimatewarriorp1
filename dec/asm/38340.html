38340: Mirrors a tile horizontally in-place.
For each scanline: reverses bits of the 3 bytes via table at reversal_table.
Called from sprites_mirror_checker, draw_snakes_sprites and head_mirroring.
INPUT: B = scanline count. DE = tile address.
OUTPUT: DE advanced past tile. Tile mirrored in-place. DESTROYS: AF, BC, HL.
mirror_tile_inplace 38340 LD B,21 B=21h: full tile height.
This entry point is used by the routine at draw_snakes_sprites.
mirror_tile_inplace_0 38342 LD H,91 HL base = bit-reversal table (reversal_table).
mirror_tile_inplace_1 38344 LD A,(DE) A = byte 0 index from tile.
38345 LD L,A HL points to reversed byte 0.
38346 LD C,(HL) C = reversed byte 0.
38347 INC DE Next: byte 1 index.
38348 LD A,(DE)
38349 LD L,A HL points to reversed byte 1.
38350 LD A,(HL) A = reversed byte 1.
38351 LD (DE),A Write reversed byte 1 in place.
38352 INC DE Next: byte 2 index.
38353 LD A,(DE)
38354 LD L,A HL points to reversed byte 2.
38355 LD A,(HL) A = reversed byte 2.
38356 DEC DE Back to byte 0 position.
38357 DEC DE
38358 LD (DE),A Write reversed byte 2 to byte 0 position.
38359 INC DE
38360 INC DE Advance to byte 2 position.
38361 LD A,C Recover reversed byte 0 (in C).
38362 LD (DE),A Write it to byte 2 position.
38363 INC DE Advance DE to next scanline.
38364 DJNZ mirror_tile_inplace_1 Repeat for B scanlines.
38366 RET Return.
