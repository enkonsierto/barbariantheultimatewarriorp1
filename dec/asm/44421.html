44421: Movement animation pointer table.
40 entries (0-39), 2 bytes each. Points to animation chains in barb_ptr_table.
Array of 2-byte LE pointers indexed by movement ID.
Each pointer targets the frame sequence for that movement.
Read by set_char_moves after updating IY+6 (movement ID).
Result stored in IY+4,IY+5 as frame sequence start pointer.
