37852: Round timer tick: decrement, refresh HUD and play sound.
Decrements timer_flag by 1, refreshes the timer display via update_time_display and plays the tick sound via sound_round_end.
Called once per loop iteration by p2_death_score_update and p1_death_score_update while draining the remaining time as bonus score.
INPUT: None. OUTPUT: Z set if the timer has reached 0, NZ if time remains. DESTROYS: AF.
timer_tick 37852 DEC A Load timer value from timer_flag and decrement by 1.
37853 LD (46808),A Write updated timer value back to timer_flag.
37856 PUSH AF Preserve decremented value and flags across the HUD and sound calls.
37857 CALL update_time_display Refresh the timer display via update_time_display.
37860 CALL sound_round_end Play timer tick sound via sound_round_end.
37863 POP AF Restore timer value and flags.
37864 OR A Set Z if timer reached 0, NZ if time still remains.
37865 RET Return.
