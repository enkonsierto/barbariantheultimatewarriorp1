46454: Active Background index.
0=Background 1 (graphics stage_1_bg, attrs bg_1_attr_data), 1=Background 2 (graphics stage_2_bg, attrs bg_2_attr_data).
Reset to 0 by game_reset at the start of each new round.
Toggled via XOR 1 by change_stage_ptr when the round timer expires.
Read exclusively by change_stage_ptr to select the active Background pointers.
active_bg 46454 DEFB 0 Active Background index. 0=Background 1, 1=Background 2.
