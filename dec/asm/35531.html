35531: Hit processor, score award and sound dispatcher.
Called once per frame from the game loop. Reads the pending combat event from pending_sound_index.
If 0, returns immediately. Events 5-6 (guard) skip the visual/score logic and jump directly to sound dispatch.
Events 1-4 (hit): draws a hit indicator glyph (26216) near the attacker column if the frame is in hit_indicator_frame_table, awards score if the frame is in score_award_frame_table, then clears pending_sound_index and plays the sound for the event.
Sound index: 1=sound_kick, 2=sound_tackle, 3=sound_slash, 4=sound_head_cut, 5=sound_guard_1, 6=sound_guard_2.
