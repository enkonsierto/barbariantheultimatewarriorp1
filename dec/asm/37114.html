37114: Generate right-edge column mask for partial tile blending.
Called from right_edge_renderer for non-0xFF tile bytes at right tile boundary.
Counts trailing 1-bits from bit 0 (LSB) to find first transparent column, loads LSB-packed mask (1,3,7,15,31,63,127) into A and jumps to mask_and_blend.
INPUT: A = right-edge tile byte, HL = framebuffer address, DE = tile source.
OUTPUT: A = pixel mask or 0 (fully transparent). DESTROYS: A, C.
right_mask_gen 37114 LD C,A C = right-edge tile byte. Tests bit 0: set means at least the rightmost pixel column is visible.
37115 BIT 0,C Bit 0 clear: byte is 0x00, no visible columns - jump to mask_byte_clear.
37117 JR Z,mask_byte_clear
37119 LD A,1 Bit 0 set. A = 0x01 (00000001b); test bit 1.
37121 BIT 1,C Bit 1 clear: 1 visible column - jump to mask_and_blend with mask 0x01.
37123 JR Z,mask_and_blend
37125 LD A,3 A = 0x03 (00000011b); test bit 2.
37127 BIT 2,C Bit 2 clear: 2 visible columns - jump to mask_and_blend with mask 0x03.
37129 JR Z,mask_and_blend
37131 LD A,7 A = 0x07 (00000111b); test bit 3.
37133 BIT 3,C Bit 3 clear: 3 visible columns - jump to mask_and_blend with mask 0x07.
37135 JR Z,mask_and_blend
37137 LD A,15 A = 0x0F (00001111b); test bit 4.
37139 BIT 4,C Bit 4 clear: 4 visible columns - jump to mask_and_blend with mask 0x0F.
37141 JR Z,mask_and_blend
37143 LD A,31 A = 0x1F (00011111b); test bit 5.
37145 BIT 5,C Bit 5 clear: 5 visible columns - jump to mask_and_blend with mask 0x1F.
37147 JR Z,mask_and_blend
37149 LD A,63 A = 0x3F (00111111b); test bit 6.
37151 BIT 6,C Bit 6 clear: 6 visible columns - jump to mask_and_blend with mask 0x3F.
37153 JR Z,mask_and_blend
37155 LD A,127 A = 0x7F (01111111b); bits 6..0 all set: 7 visible columns.
37157 JR mask_and_blend Jump to mask_and_blend with mask 0x7F.
