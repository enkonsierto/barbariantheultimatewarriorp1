36037: String line renderer.
Used by the routines at main_menu, one_player_mode, key_assign and two_player_mode.
Renders a full line of text composed of multiple words to screen.
For each index, resolves the word address via word_addr_solver and draws it via draw_word. Stops when index value is 255.
INPUT: HL = VRAM destination address. DE = string index list address (select_option).
OUTPUT: None. DESTROYS: HL, DE.
draw_string 36037 PUSH DE Preserves the index list pointer.
36038 LD A,(DE) Reads the current word index into A.
36039 CALL word_addr_solver Resolves word index to character data address in DE.
36042 CALL draw_word Draws the word. HL advances to the next free column.
36045 POP DE Restores the index list pointer.
36046 INC DE Advances to the next index.
36047 LD A,(DE) Reads the next index.
36048 CP 255 Tests for end-of-line sentinel (255).
36050 JR NZ,draw_string Loops back if more words remain.
36052 RET Return.
