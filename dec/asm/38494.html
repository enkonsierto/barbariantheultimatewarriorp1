38494: Interrupt Vector Table Generator
Used by the routine at game_start.
Will generate all the 257 bytes of the interrupt table where, in loading time, are the game start routines never used later.
INPUT: C = 240, B = 0, HL = 48896 (game_start)
i_v_t_generator 38494 LD (HL),C Copy 240($F0) to the future Vector Table (game_start).
38495 INC HL Increments in 1 HL
38496 DJNZ i_v_t_generator Decrements B and compare with 0 (this loop will execute 256 times).
38498 LD (HL),C After copying the 256 bytes, we need to copy F0 once more (257 times).
38499 RET Returns to 35244
