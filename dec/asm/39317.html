39317: Snake animation phase stepper.
Used by the routine at draw_snakes_sprites.
Advances the draw phase index in the high nibble of IY+7 by 1 each call, leaving the current energy value in the low nibble intact.
When the index reaches 14 the animation cycle is complete and it resets to 0 (idle). Uses the resulting index to resolve a pointer to the corresponding snake sprite frame via the table at snake_pointer_table.
IY+7 byte layout: bits 7-4 draw phase (0 idle, 1-13 active, 14->0 wrap); bits 3-0 current energy value (0-13).
INPUT: IY = character state block (p1_controller_type P1 or p2_controller_type P2).
OUTPUT: A = first byte of the snake sprite frame. DE = pointer to the next byte.
DESTROYS: AF, BC, DE, HL.
