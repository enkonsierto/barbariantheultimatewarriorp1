38412: Flush framebuffer to screen.
Used by the routine at entry_point.
Copies 89 scanlines (28 bytes each) from the RAM framebuffer at framebuffer to the ZX Spectrum VRAM, using the pre-built VRAM address table at left_col_addr_table.
SP is redirected to 23560 (left_col_addr_table+8), stepping past the top address so only the 89 game-area rows are blitted. 28 unrolled LDI instructions copy each row without inner-loop overhead.
Counterpart to copy_bg_to_frmbuff, which fills the framebuffer each frame from the active stage background.
INPUT: None. OUTPUT: None. DESTROYS: All registers. SP: is saved to saved_sp and restored on exit.
