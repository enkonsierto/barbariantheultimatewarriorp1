33876: Selects the next pointer and the note.
Reads the next note from a music sequence and updates the pointer.
If it finds the end marker (64), it restarts the sequence in a loop.
INPUT: HL = address of sequence pointer (melody_ptr or bass_ptr).
OUTPUT: HL = next pointer, A = note value.
DESTROYS: DE.
select_music_pointer 33876 LD E,(HL) Loads the current note address into DE (low and high byte).
33877 INC HL
33878 LD D,(HL)
33879 INC DE DE now points to the next note.
This entry point is used by the routine at restart_music.
select_music_pointer_0 33880 LD A,(DE) Reads the note value into A.
33881 CP 64 Compares with 64 (end of sequence marker).
33883 JR Z,restart_music If 64, jumps to restart_music to restart.
33885 LD (HL),D Saves the new pointer (updates position in sequence).
33886 DEC HL
33887 LD (HL),E
33888 RET Returns with note in A.
