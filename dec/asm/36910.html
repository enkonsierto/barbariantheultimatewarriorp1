36910: Tile render engine.
Used by the routines at draw_single_tile, render_head_tile and shadow_head.
Reads one tile control byte from the source address and dispatches to one of four renderers based on its bit fields.
Control byte: bit 7 = composite tile flag; bits 1-0 = render mode (00 direct copy, 11 AND-mask overlay, 01/10 edge mask); bit 6 = mirror flag, swaps left and right edge renderers when set.
INPUT: DE = tile data address. HL = framebuffer destination address. OUT: None. DESTROYS: AF, B, C.
tile_render_engine 36910 LD A,(DE) Reads the control byte from the tile source and advances the address.
36911 INC E
36912 BIT 7,A Bit 7 set: composite tile - dispatches to comp_tile_dispatcher.
36914 JP NZ,comp_tile_dispatcher
36917 LD C,A Isolates bits 1-0; dispatches to AND-mask overlay (tile_renderer) if bits 1-0 are 11.
36918 AND 3
36920 CP 3
36922 JP Z,tile_renderer
36925 OR A Dispatches to direct-copy renderer (direct_copy_tile_renderer) if bits 1-0 are 00.
36926 JP Z,direct_copy_tile_renderer
36929 LD A,C Restores the control byte; if the mirror flag (bit 6) is set, flips bits 1-0 to swap left and right edge renderers.
36930 BIT 6,A
36932 JR Z,tile_render_engine_0
36934 XOR 3

Left-edge AND-mask renderer loop (21 scanlines).
Outer loop for the left-edge AND-mask tile renderer. Calls left_edge_renderer 21 times (B=21), once per scanline of the tile.
