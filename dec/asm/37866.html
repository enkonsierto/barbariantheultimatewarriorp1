37866: Decapitation setup.
Called from sprite_dispatcher when the sprite dispatcher encounters frame index 17.
IY points to the attacker state block on entry.
Identifies the defender (opposite character) and applies movement 37 (decapitation fall) to them via set_char_moves. Then, if no head is already in flight (head_state_flag != 1), computes the head spawn position in the framebuffer from the attacker X tile column plus a 15-row vertical offset, and initialises the flying-head state block at head_state via head_init.
IN: IY attacker state block (p1_controller_type or p2_controller_type).
OUT: None. head_state initialised if head_state_flag != 1.
DESTROYS AF, AF', IY.
decap_setup 37866 PUSH AF Preserve AF (frame index that triggered this call).
37867 PUSH IY Preserve IY (attacker state block).
37869 LD A,IYl A = attacker type: low byte of IY base address (89=P1, 98=P2).
37871 LD IY,46425 Default: IY = P1 state block R46425 (defender if attacker is P2).
37875 CP 89 Is attacker P1 (type 89)?
37877 JR NZ,decap_setup_0 No (attacker is P2) -> defender is P1, keep IY = p1_controller_type.
37879 LD IY,46434 Yes (attacker is P1) -> defender is P2, set IY = p2_controller_type.
decap_setup_0 37883 LD A,37 A = 37: movement code for decapitation fall.
37885 CALL set_char_moves Apply movement 37 to the defender via set_char_moves.
37888 POP IY Restore IY: now points to the attacker block.
37890 LD A,(46810) Read head active state from head_state_flag.
37893 CP 1 Is head already in flight (state = 1)?
37895 JR Z,decap_setup_3 Yes -> skip initialisation, jump to decap_setup_3.
37897 LD A,(IY+3) A = attacker X tile position (IY+3).
37900 LD L,A L = column offset in framebuffer.
37901 LD H,94 H = 0x5E: framebuffer base framebuffer.
37903 LD DE,480 DE = 480 (15 rows * 32): vertical offset to head spawn row.
37906 ADD HL,DE HL = framebuffer address at attacker column, 15 rows down.
37907 LD A,(IY+2) A = mirror flags byte (IY+2).
37910 AND 1 Isolate bit 0: mirror flag (1 = attacker facing left).
37912 JR Z,decap_setup_1 Not mirrored -> head spawns directly at attacker column, no adjustment needed.
37914 LD DE,6 DE = 6: hardcoded horizontal offset for head spawn position (mirrored sprites only).
37917 ADD HL,DE Shift head spawn point 6 bytes right.
decap_setup_1 37918 EX AF,AF' Save mirror bit to AF', free A for next operations.
37919 XOR A A = 0.
37920 LD (46825),A Write 0 to unused_buffer (redundant: always 0, never read from this buffer).
37923 LD DE,47017 DE = normal_traj_data: 18-entry head flight trajectory table (Dx,Dy pairs per frame).
37926 LD BC,46853 BC = p2_head_tile_seq: P2 head tile sequence (default, used when attacker is P2).
37929 LD A,IYl A = attacker type (IY+0 low byte).
37931 CP 98 Is attacker P2 (type 98)?
37933 JR Z,decap_setup_2 Yes -> keep BC = p2_head_tile_seq (P2 head tiles), jump to decap_setup_2.
37935 LD BC,46835 No (attacker is P1) -> BC = R46835 (P1 head tiles).
decap_setup_2 37938 EX AF,AF' Restore A = mirror bit from AF'.
37939 CALL head_init Initialise flying-head state block head_state via head_init.
decap_setup_3 37942 POP AF Restore AF.
37943 RET Return.
