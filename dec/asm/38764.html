38764: Single scanline clear with attribute wipe.
Clears one pixel scanline byte and sets the attribute to black (0) for each of the 28 tiles in a row. Called 8 times per tile row by clear_row to erase all pixel data of a full row. single_scanline_clear also shares the inner loop from 38767 with B=26 to preserve the leftmost snake column.
INPUT: HL = VRAM address of the scanline byte to clear.
OUTPUT; HL = restored to the entry value. DESTROYS: A, B, DE.
clear_scanline_ink_attr 38764 PUSH HL Preserve the entry VRAM address.
38765 LD B,28 B=28, one iteration per tile column across the full game window row.
This entry point is used by the routine at single_scanline_clear.
clear_scanline_ink_attr_0 38767 XOR A A=0, value to clear with.
38768 LD (HL),A Clear the current VRAM scanline byte.
38769 CALL get_attr_address
38772 LD E,L Form the full attribute address in DE using A and the column byte from L.
38773 LD D,A
38774 XOR A A=0, black attribute value.
38775 LD (DE),A Write black attribute (ink=0, paper=0) to this tile.
38776 INC HL Advance to the next tile column.
38777 DJNZ clear_scanline_ink_attr_0 Repeat for all tile columns.
38779 POP HL Restores the caller's HL value.
38780 RET Return.
