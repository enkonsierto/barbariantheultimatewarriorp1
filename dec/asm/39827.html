39827: Per-frame character update.
Called from combat_loop for each fighter with IY pointing to their state block.
Returns immediately if IY+0=0 (inactive) or frame_counter_hit non-zero (3-frame cycle skip, see frame_counter_hit_inc).
INPUT: IY = character state block. OUTPUT: None. DESTROYS: AF, BC, HL.

39830 OR A Return if zero (inactive).
39831 RET Z
39832 LD A,(47475) Load frame_counter_hit (hit-detection skip counter)
39835 OR A Return if non-zero (frame skip).
39836 RET NZ
