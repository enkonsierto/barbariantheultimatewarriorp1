36706: Menu rainbow colour animation.
Cycles 7 times (112 ms each), incrementing ink tint across 7 menu rows: white -> yellow -> cyan -> green -> magenta -> green -> cyan (pause/quit conditional).
rainbow_animation 36706 LD B,7 Set a counter of 7 loops.
rainbow_animation_0 36708 PUSH BC Preserves the counter.
36709 CALL waits_112ms Waits 112 ms.
36712 LD A,7 Target ink colour: white (7).
36714 LD HL,22853 First attribute row base address for this colour step.
36717 LD BC,5377 B=21 tiles per row, C=1 row to step.
36720 CALL step_ink_up Advance ink colour toward the target value via R38664.
36723 DEC A Target ink colour: yellow (6).
36724 LD HL,22918 Set the initial screen address to colour.
36727 CALL step_ink_up Increments in 1 the attribute ink.
36730 ADD HL,DE Selects the next line below.
36731 DEC A Target ink colour: cyan (5).
36732 CALL step_ink_up Increments in 1 the attribute ink.
36735 ADD HL,DE Selects the next line below.
36736 DEC A Target ink colour: green (4).
36737 CALL step_ink_up Increments in 1 the attribute ink.
36740 ADD HL,DE Selects the next line below.
36741 DEC A Target ink colour: magenta (3).
36742 CALL step_ink_up Increments in 1 the attribute ink.
36745 ADD HL,DE Selects the next line below.
36746 INC A Target ink colour: green (4).
36747 CALL step_ink_up Increments in 1 the attribute ink.
These two attribute writes only fire if quit_scancode quit and pause_scancode pause scancodes are still 0 not yet assigned. In practice both are non-zero during normal gameplay, so these rows are never coloured at runtime. They would only activate before any key assignment has occurred.
