35765: Character AI engine.
Called by input_ai_dispatcher with A = extra state byte (p2_frame_index for P1, p1_frame_index for P2) and IY pointing to the active character state block.
Performs a 4-step lookup chain:
1. extra state -> AI profile block (ai_profile_0) via pointer table ai_profile_table.
2. Fighter distance (fighter_distance) -> profile column 0-7 via second_ai_state_table.
3. Profile column -> movement block ID (0-22) -> 8-byte row in ai_movement_table (x8 offset).
4. R register (pseudo-random, masked 0-31) + fight counter (fight_counter) -> difficulty entry (ai_difficulty_table) -> subindex (0-7) -> movement ID from ai_movement_table.
Maps the movement ID to a key-state bitmask and stores it in IY+1 so the movement engine sees it as real input.
IN A = extra state byte. IY = character state block.
OUT (IY+1) = AI-generated key state bitmask.
DESTROYS AF, BC, DE, HL.

35768 CALL solve_pointer Resolve AI profile pointer via solve_pointer; HL = profile block address in ai_profile_0.

35772 LD A,(46834) A = fighter distance fighter_distance.

35785 LD DE,8 DE = 8 (byte stride per block in ai_movement_table).
35788 CALL bin_multiply HL = A * 8 via bin_multiply; byte offset into the movement selection table.
35791 LD DE,47974 DE = movement table base ai_movement_table.

35795 PUSH HL Preserve movement block pointer.
35796 LD A,R A = R register; pseudo-random source.
35798 AND 31 Mask A to 0-31; random index into the difficulty entry.
35800 PUSH AF Preserve random value.
35801 LD A,(47558) A = fight counter fight_counter (0-7); difficulty level index.
35804 LD HL,47559 HL = difficulty table base ai_difficulty_table.
35807 CALL solve_pointer Resolve difficulty entry pointer for this fight level via solve_pointer.
35810 POP AF Restore random value.
35811 CALL get_byte_value Index into difficulty entry using random value via get_byte_value; A = subindex (0-7).
35814 POP HL Restore movement block pointer.
35815 CALL get_byte_value Read movement ID from block at subindex A via get_byte_value; A = movement ID.
35818 CALL input_mirror_correcter Mirror movement ID if necessary, via input_mirror_correcter (input-class table input_lookup_table, mirror-corrected for IY+2 bit 0).
35821 LD (IY+1),A Store AI key bitmask in (IY+1).
