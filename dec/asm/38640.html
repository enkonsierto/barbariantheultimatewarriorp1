38640: Fade ink colour to black.
Used by the routine at menu_fade_out.
For each of C rows, reads the ink bits of each attribute byte via AND 7 and decrements the ink value by 1 unless it is already 0. Processes B tiles per row, then advances HL 32 bytes to the next attribute row.
Called from menu_fade_out to progressively darken the game window during the fade-out transition.
INPUT: HL = first attribute address (top-left tile). B = tiles per row (26). C row count (3).
OUTPUT: None. DESTROYS A, C, DE, HL.
