49046: Per-frame horizontal X-delta table
Indexed by IY+8 (current frame index) from x_pos_controller. Each byte is a signed 8-bit displacement added to IY+3 (X tile). Positive values move right, negative values move left. The mirror flag (IY+2 bit 0) negates the value so mirrored characters always move in the correct direction.
