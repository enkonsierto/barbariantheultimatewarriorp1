36879: Calculates the VRAM address of the scanline above.
Given a VRAM scanline address in HL, steps back one scanline, handling tile-row and screen-third boundaries in the non-linear ZX Spectrum screen layout.
INPUT: HL = current VRAM scanline address. OUTPUT: HL = address of the scanline above. DESTROYS: A, F.
calc_prev_scanline_addr 36879 DEC H Steps back one scanline (H = $100, 256 bytes).
36880 LD A,H Checks whether the decrement crossed a tile boundary (lower 3 bits of H become 7 when wrapping backward).
36881 CPL
36882 AND 7
36884 RET NZ Within the same tile; address is ready.
36885 LD A,L Moves up one tile row (L encodes the tile row; subtract 32 to reach the row above).
36886 SUB 32
36888 LD L,A
36889 RET C Tile-row underflow carried into the screen-third; address is ready.
36890 LD A,H No carry; the screen-third was over-decremented - adds 8 to H to correct it.
36891 ADD A,8
36893 LD H,A
36894 RET Return.
