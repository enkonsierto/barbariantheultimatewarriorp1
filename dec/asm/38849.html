38849: Blit stage background to screen row by row.
Used by the routine at entry_point.
Copies backfround tile graphics and RLE colour attributes from the active background source to VRAM one row at a time, inserting a 112 ms pause after each of the first 15 rows to produce a visible top-to-bottom reveal. The final 2 rows are blitted without a pause. Resets the attribute stream pointer active_bg_attr_ptr to its initial value on exit so the next call starts from the beginning of the attribute table.
INPUT: None. OUTPUT: None. DESTROYS: AF, BC, DE, HL.
paints_stage_screen 38849 LD HL,(46461) Load the initial attribute stream address from bg_1_attr_data and preserve it on the stack for the reset at the end.
38852 PUSH HL
38853 LD DE,(46459) Load the active stage graphics source address from active_bg_bitmap_ptr into DE (stage_1_bg stage 1, stage_2_bg stage 2).
38857 LD HL,16610 Set HL to the VRAM address of the first tile of the game window.
38860 LD B,15 B=15, one iteration per row for the top section; the last 2 rows are handled separately without a pause.
paints_stage_screen_0 38862 PUSH BC Preserve the loop counter.
38863 CALL copy_bg_row Copy one full tile row (28 tiles, 8 scanlines) from source to VRAM via copy_bg_row; DE and HL advance to the next row.
38866 CALL apply_bg_attr_row Apply RLE colour attributes to the current VRAM row via apply_bg_attr_row; active_bg_attr_ptr advances to the next attribute entry.
38869 CALL waits_112ms Wait 112 ms via waits_112ms.
38872 POP BC Restore the loop counter.
38873 DJNZ paints_stage_screen_0 Repeat for all 15 rows.
38875 LD HL,22466 Advance DE and HL to the penultimate row.
38878 CALL apply_bg_attr_row Copy the penultimate row via copy_bg_row.
38881 CALL waits_112ms Apply attributes to the penultimate row via apply_bg_attr_row.
38884 LD HL,22501 Advance DE and HL to the last row.
38887 CALL apply_bg_attr_row Copy the last row via copy_bg_row.
38890 CALL waits_112ms Apply attributes to the last row via apply_bg_attr_row.
38893 POP HL Restore initial attribute stream address from the stack.
38894 LD (46461),HL Write restored address to active_bg_attr_ptr, resetting the attribute stream pointer for the next call.
38897 RET Return.
