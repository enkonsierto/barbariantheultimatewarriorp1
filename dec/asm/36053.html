36053: Word index to address resolver.
Resolves a word index to its character data address from the string pointer table at string_pointers_table.
INPUT: A = word index. HL = VRAM address.
OUTPUT: HL = VRAM address (preserved). DE = word data address. DESTROYS: None.
word_addr_solver 36053 PUSH HL Preserves the VRAM address.
36054 LD HL,48737 Loads the string pointer table base string_pointers_table into HL.
36057 CALL solve_pointer Resolves the pointer for index A via solve_pointer; result in HL.
36060 POP DE Recovers the VRAM address into DE.
36061 EX DE,HL Swaps registers: HL=VRAM address, DE=word address.
36062 RET Return.
