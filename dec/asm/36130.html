36130: Main Menu loop
Used by the routine at entry_point.
Draws the main menu text and runs the rainbow colour animation loop until a valid option key (0-3) is pressed.
On selection jumps to the corresponding game mode: 0=demo, 1=1P, 2=2P, 3=developer info screen.
INPUT None. OUTPUT None (tail-calls game mode entry points).
DESTROYS AF, BC, DE, HL.
main_menu 36130 CALL sweep_transition Fade out any previous screen content to black via sweep_transition.
This entry point is used by the routines at one_player_mode, key_assign and two_player_mode.
