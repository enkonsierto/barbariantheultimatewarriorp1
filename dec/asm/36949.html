36949: Left-edge AND-mask tile renderer.
Used by the routines at tile_render_engine and tile_renderer.
Renders one scanline of a 3-byte-wide tile. Scans each byte for transparency (0xFF); the first visible byte receives a left-edge column mask via left_mask_gen, and any remaining visible bytes are written directly to the framebuffer.
Called 21 times per tile by the outer loop at left_edge_loop. On return, DE and HL point to the start of the next scanline.
INPUT: DE = tile source address (within a single 256-byte page). HL = framebuffer destination. OUTPUT: None. DESTROYS: AF, C.
