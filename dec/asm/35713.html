35713: Input and AI dispatcher.
Called once per character from combat_loop.
IY points to the active character state block. Selects between human input and AI, or returns immediately if the character is inactive. Human-controlled characters read the keyboard/joystick and store the key bitmask in IY+1. CPU-controlled characters call character_ai_engine to generate an AI key bitmask in their state block.
IN IY character state block.
OUT (IY+1) key state bitmask (human or AI).
DESTROYS AF, A. IY is preserved on return.
input_ai_dispatcher 35713 LD A,(IY+0) A = (IY+0) controller type / active flag.
35716 BIT 7,A Test bit 7: 0 = human-controlled, 1 = CPU-controlled.
35718 JR NZ,input_ai_dispatcher_0 If bit 7 set (CPU), branch to AI path at 35729.
35720 OR A Human path: A = 0 means inactive character.
35721 RET Z If inactive (IY+0 = 0), return immediately.
35722 CALL human_input_reader Call human_input_reader to read human input for this controller type.
35725 LD (IY+1),A Store key bitmask returned in A into (IY+1).
35728 RET Return (human-controlled case done).
input_ai_dispatcher_0 35729 LD A,IYl CPU path: A = (IY+1) character type (89=P1, 98=P2).
35731 CP 89 Is this P1 type (89)?
35733 JR NZ,input_ai_dispatcher_1 If not P1, branch to P2 AI path at 35750.
35735 LD A,(46442) Load P1 extra state byte from p2_frame_index (AI profile / difficulty).
35738 PUSH IY Preserve caller IY.
35740 LD IY,46425 Set IY to P1 state block base p1_controller_type.
35744 CALL character_ai_engine Call character_ai_engine to run P1 AI and write synthetic key state into IY+1.
35747 POP IY Restore original IY.
35749 RET Return (CPU-controlled P1 done).
input_ai_dispatcher_1 35750 LD A,(46433) P2 AI path: load P2 extra state byte from p1_frame_index.
35753 PUSH IY Preserve caller IY.
35755 LD IY,46434 Set IY to P2 state block base p2_controller_type.
35759 CALL character_ai_engine Call character_ai_engine to run P2 AI and write synthetic key state into IY+1.
35762 POP IY Restore original IY.
35764 RET Return (CPU-controlled P2 done).
