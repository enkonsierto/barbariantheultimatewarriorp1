47578: Secondary AI state-to-profile-column table.
40-byte table (second_ai_state_table-47617) indexed by AI state fighter_distance (range 0-39).
Each entry is a byte offset (0-7) selecting which column of the 8-byte profile block at R47798 is used to obtain the movement_block_id.
This allows the AI combat state (distance, posture, reaction) to bias the profile column independently of the extra_state selection.
Used by character_ai_engine: Call get_byte_value with HL=second_ai_state_table and A=fighter_distance.
