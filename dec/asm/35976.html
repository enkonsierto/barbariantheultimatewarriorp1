35976: Binary multiplier. Computes HL = A * DE.
Used by the routine at character_ai_engine.
General-purpose 8bits * 16bits unsigned multiply using shift-and-add.
Iterates 8 times, one per bit of A LSB first. If the current bit is set, adds DE to the HL accumulator. DE is left-shifted each iteration to track the positional value of each bit.
Called from character_ai_engine with DE=8 to compute a byte offset into the AI movement table ai_movement_table (effective result: HL = A * 8).
INPUT A = multiplicand (8-bit unsigned), DE = multiplier (16-bit unsigned).
OUTPUT HL = A * DE (16-bit result).
DESTROYS HL, DE, B.
