38936: Copy one tile row from stage background directly to screen.
Used by the routine at paints_stage_screen.
Copies all 8 scanlines of a 28-tile-wide row from the stage background graphics to the ZX Spectrum screen VRAM. Called from paints_stage_screen to paint the stage row by row during screen transitions.
INPUT: DE = stage background source address. HL = VRAM destination address, top scanline of the target tile row.
OUTPUT: HL = VRAM address of the next tile row below. DE = advanced by 224 bytes (28 bytes x 8 scanlines).
DESTROYS AF, B.
copy_bg_row 38936 LD B,8 Sets the scanline counter to 8, one iteration per scanline of the tile row.
copy_bg_row_0 38938 PUSH BC Saves the loop counter and the address for the current scanline.
38939 PUSH HL
38940 CALL copy_bg_scanline Copies one 28-byte scanline from the stage background to VRAM via copy_bg_scanline.
38943 POP HL Restores the VRAM address to the start of the current scanline.
38944 CALL calc_next_scanline_addr Advances the VRAM pointer to the next scanline via calc_next_scanline_addr.
38947 POP BC Restores the loop counter.
38948 DJNZ copy_bg_row_0 Repeats for all 8 scanlines of the tile row.
38950 RET Returns once the full tile row has been copied.
