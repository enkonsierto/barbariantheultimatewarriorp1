46817: Player 2 overlay state block.
9-byte block reusing the same layout as the head state block at head_state, allowing render_head_tile and head_mirroring to serve both renderers without modification.
Written each frame by p2_overlay_tile before jumping to render_head_tile.
p2_overlay_block_state 46817 DEFB 0 IX+0: mirror flag (0=facing right, 1=facing left). Copied from p2_mirror_flags bit 0.
46818 DEFB 1 IX+1: not written by p2_overlay_tile; not read.
46819 DEFW 47219 IX+2,3: pointer to current tile index entry = p2_overlay_tile_table + P2 frame index.
46821 DEFW 0 IX+4,5: not used by P2 overlay path. Retains head state values.
46823 DEFW 24074 IX+6,7: framebuffer destination address. Computed by p2_overlay_tile.
