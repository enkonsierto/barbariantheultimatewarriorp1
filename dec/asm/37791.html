37791: P2-death score update and time drain loop.
Called by p1_wins_seq (P1-wins sequence) after P2 has been defeated.
MODE 1P/DEMO (timer_flag=255): jumps immediately to post_fight_handler end-of-match handler without touching the score. The timer is never decremented.
MODE 2P (timer_flag=0..90): awards 10 points to P1 score (p1_score) and refreshes the score display via paint_score, then calls timer_tick to decrement the timer by 1 and update the HUD. Loops until timer_tick returns Z (timer_flag has reached 0), then falls through to post_drain_wait.
Each iteration awards +10 to P1 and drains 1 timer unit, so the total bonus equals remaining_time * 10.
p2_death_score_update 37791 LD A,(46808) Read the round timer (timer_flag).
37794 CP 255 Is it 255 (1P/demo mode - infinite time)?
37796 JP Z,post_fight_handler Yes - jump to post_fight_handler end-of-match handler without scoring.
p2_death_score_update_0 37799 LD HL,(47429) HL = current P1 score (p1_score).
37802 LD DE,10 DE = 10 points to add.
37805 ADD HL,DE HL = score + 10.
37806 LD (47429),HL Write updated score back to p1_score.
37809 CALL paint_score Refresh P1 score display via paint_score.
37812 CALL timer_tick Decrement timer by 1 and update HUD via timer_tick. Returns Z if timer=0.
37815 JR NZ,p2_death_score_update_0 Timer not zero - loop back to award another +10.
37817 JR post_drain_wait Timer exhausted - fall through to post_drain_wait.
