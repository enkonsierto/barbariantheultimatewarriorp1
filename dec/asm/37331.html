37331: Direct copy tile renderer.
Used by the routine at tile_render_engine.
Copies tile data directly to the framebuffer without masking: 3 bytes (24 pixels) wide, B scanlines high.
Framebuffer advances 32 bytes per row; source DE stays in single 256-byte page.
INPUT: B scanline count (21 full tile or sub-tile height). DE tile source address (low byte only advanced). HL framebuffer destination.
OUTPUT: None. DESTROYS: AF,BC,DE,HL.
direct_copy_tile_renderer 37331 LD B,21 Sets B=21 for full tile height (entry from R36910).
This entry point is used by the routine at comp_tile_dispatcher.
