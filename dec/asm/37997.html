37997: Flying head animator.
Called once per frame from the main game loop via 35319.
Moves and renders the severed head using the 8-byte state block at head_state, addressed via IX.
Movement control data: 2-byte pairs (control byte, magnitude). Bit 0 = move up (move_scanl_up), bit 1 = move down (move_scanl_down). Pointer at IX+4,5 advances 2 bytes per frame.
Tile sequence at IX+2,3 advances 1 byte per frame. When the byte value is 255, INC A wraps to 0 and triggers mark_head_landed: sets head_state_flag=255 (landed), unblocking goblin spawn in goblin_entry.
After horizontal movement, column byte (L AND 31) is checked against 0, 1 and 29. Any edge column triggers deactivate_head: sets head_state_flag=0 (fully inactive).
Landed path: head_state_flag=255, INC A wraps to 0, JR Z jumps to render_head_tile skipping all movement logic. The head is redrawn in place each frame until goblin_kicks re-activates it.
