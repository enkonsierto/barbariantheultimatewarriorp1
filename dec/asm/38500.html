38500: Interrupt Service Routine
Triggered by Z80 interrupt mode 2 (IM 2).
In execution time, this routine is written by 49031 to 0xF0F0/61680 (more info in 61569) so the Vector table can call it.
Two responsibilities: frequency divider via isr_counter, combat timer decrement via timer_flag.
OUTPUT: isr_counter incremented or reset; timer_flag decremented.
int_serv_routine 38500 PUSH AF Preserve AF, BC
38501 PUSH BC
38502 LD A,(47474) isr_counter (ISR frame counter, cycles 0-8).
38505 INC A Advance counter.
38506 CP 9 Has the counter reached 9 (one timer tick)?
38508 JR Z,int_serv_routine_0 Yes: reset counter and process timer.
38510 LD (47474),A No: store updated counter and exit ISR.
38513 JR int_serv_routine_1 Jump to exit.
int_serv_routine_0 38515 XOR A Counter reached 9 and is reset to 0.
38516 LD (47474),A
38519 LD A,(46808) timer_flag (combat timer: 90->0 in 2P mode; 255 = infinite/1P).
38522 CP 255 Is it infinite-time mode (1P/Demo)?
38524 JR Z,int_serv_routine_1 Yes: skip decrement, exit.
38526 OR A Is timer already 0?
38527 JR Z,int_serv_routine_1 Yes: already expired, skip decrement, exit.
38529 LD B,A No: preserve timer value in B.
38530 LD A,(46443) (goblin active flag: 0=inactive, 255=active). Is the goblin currently active?
38533 CP 255
38535 LD A,B Restore timer value into A.
38536 JR Z,int_serv_routine_1 Goblin active: freeze timer, exit without decrementing.
38538 DEC A Decrement combat timer by 1.
38539 LD (46808),A Store updated timer value back to timer_flag.
int_serv_routine_1 38542 POP BC Restore BC and AF.
38543 POP AF
38544 EI Re-enable interrupts.
38545 RETI Return from interrupt.
