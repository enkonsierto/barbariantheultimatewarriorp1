39968: Tackle cross-side animation controller.
Used by the routine at entry_point.
Two-phase sequencer for roll/jump-over side-swap. Triggered by x_update_and_tackle_gate when one fighter rolls (mv17 frame 38) while the other jumps over (frames 8-9).
Gated by frame_counter_hit (3-frame cycle via frame_counter_hit_inc) and tackle_overlap_flag (phase selector).
Phase 1 (tackle_overlap_flag=1): apply mv34 (changeside) to both, advance to phase 2.
Phase 2 (tackle_overlap_flag=2): wait for mid-swap frame (P1 IY+8=16), apply mv0 + mirror flip + X correction (+-2 tiles).
Called pre-HALT at 35325; reads frame_counter_hit set by previous frame's frame_counter_hit_inc (post-HALT).
INPUT: None. OUTPUT: None. DESTROYS: AF, BC, DE, HL, IY.
