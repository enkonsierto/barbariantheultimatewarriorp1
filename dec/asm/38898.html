38898: Apply RLE colour attributes to a screen row.
Used by the routine at paints_stage_screen.
Reads the RLE attribute stream from active_bg_attr_ptr and writes colour attributes. The stream is a sequence of count+value byte pairs. Each pair applies one attribute byte to the next <count> consecutive tiles. A count of 255 signals the end of the row.
After the row is processed, active_bg_attr_ptr is advanced past the end-of-row marker, ready for the next row.
INPUT: HL = VRAM address of any scanline within the target screen row. active_bg_attr_ptr active stage attribute stream pointer. OUTPUT: None. DESTROYS: AF, BC, DE, HL, alternate registers.

Stream format: count+value byte pairs. The first byte of each pair is the number of consecutive tiles to colour; the second is the attribute value to apply. A count of 255 ends the row.
