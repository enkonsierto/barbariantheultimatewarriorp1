39156: Input-gated movement transition.
Called every frame from char_frame_update. Runs through three gates before attempting a movement change; returns early if any fails:
1. Current movement is non-interruptible (bit 6 of its properties byte in 44755).
2. A special event is active (tackle_overlap_flag != 0, e.g. tackle or collision).
3. Current frame is non-controllable (IY+2 bit 7 set).
If all gates pass, mirror-corrects the input via input_mirror_correcter, resolves a candidate movement via moves_solver, and returns unchanged if the result is 255 or matches the current movement. On a valid new movement, applies it via set_char_moves and rewinds the frame pointer by 1 so the animation starts at its first real frame.
INPUT: IY = character state block (p1_controller_type or p2_controller_type).
OUTPUT: None. Updates IY+6 (movement ID), IY+4/IY+5 (frame pointer) and IY+2 (mirror flag) via set_char_moves. DESTROYS: AF, HL.
