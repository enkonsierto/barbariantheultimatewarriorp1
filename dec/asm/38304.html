38304: Generic 16-bit pointer lookup.
Used by the routines at change_stage_ptr, character_ai_engine, word_addr_solver, set_char_moves, sprite_dispatcher, moves_solver and update_snake_frame_id.
Reads a pointer from a table given a index in A. Adds A*2 to HL and reads the 2-byte entry.
INPUT: A = index. HL = table base address. OUTPUT: HL = resolved pointer. DESTROYS: AF, DE, HL.
solve_pointer 38304 PUSH DE Save table base (HL) on the stack.
38305 PUSH HL
38306 LD L,A HL = A * 2 (byte offset into the pointer table; each entry is 2 bytes).
38307 LD H,0
38309 ADD HL,HL
38310 POP DE DE = table base.
38311 ADD HL,DE HL = table base + offset, points to the target entry.
38312 LD E,(HL) DE = 16-bit pointer from table entry.
38313 INC HL
38314 LD D,(HL)
38315 POP HL Restore HL (table base).
38316 EX DE,HL Swap DE and HL: HL = resolved pointer.
38317 RET Return.
