37631: End-of-fight dispatcher.
Used by the routine at x_pos_controller.
Called when the loser's X tile reaches 30 (off-screen, "pulled" by the goblin).
Checks which player has won through checking who has frame 19 (victory) in IY+6, then dispatches to the appropriate end-of-fight sequence.
end_of_fight_dispatcher 37631 LD IY,46425 IY = player 1 state block p1_controller_type.
37635 LD A,(IY+6) Load movement ID (IY+6).
37638 CP 19 Is movement 19 (victory)?
37640 JR NZ,end_of_fight_dispatcher_0 No - jump to end_of_fight_dispatcher_0 to check time/P2 (P1 has died).
37642 CALL p1_wins_seq Yes - P1 wins: call p1_wins_seq (score update, victory sequence (P2 has died).
37645 JP reset_p2_state_block Jump to reset_p2_state_block to reinitialise P2 block for next round.
end_of_fight_dispatcher_0 37648 LD A,(46808) Load round timer value from timer_flag.
37651 CP 255 Is time = 255 (non-timed game = 1P game)?
37653 JP Z,game_reset Yes - Game over. jump to game_reset back to menu without scoring.
37656 LD IY,46434 No (It's a 2 players game); IY = P2 state block p2_controller_type.
37660 CALL p2_wins_seq Call p2_wins_seq to run P2-wins sequence (P1 has died, timed 2P mode).
37663 JP reset_p1_state_block Jump to reset_p1_state_block to reinitialise P1 block for next round.
