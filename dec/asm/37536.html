37536: Animation frame stepper.
Used by the routines at combat_loop and char_frame_update.
Advances the animation frame pointer for the character pointed by IY, fires events for specific frame indices, and processes end-of-sequence sentinels.
Frames 20 and 33 trigger a goblin spawn check via goblin_entry. Frame 87 (dying) plays the tackle sound via sound_tackle (The winner kicks the loser). Sentinel 253 restarts the current movement via set_char_moves, 254 holds on the last frame, and 255 chains to the next queued animation.
INPUT: IY character state block.
OUTPUT: IY+4,IY+5 updated frame pointer. IY+8 new frame index.
DESTROYS: AF, DE.
