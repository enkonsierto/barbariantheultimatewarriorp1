35414: Sets infinite time or not.
Sets the timer depending if there are two or one (human) players.
Called on each game restart via 35288. If timer_flag is already 255 (initial value on load, or 1P/demo mode), returns immediately.
Otherwise writes 101 to timer_flag for 2P mode: the extra counts above 90 compensate for the timer decrement between menu selection and game start, so the HUD reads 90 when combat begins.
Subsequent rounds bypass this routine and load 90 directly via 36519.
set_infinite_time 35414 LD A,(46808) Load round timer timer_flag into A.
35417 CP 255 If 255, already in infinite time mode; return.
35419 RET Z Return.
35420 LD A,101 A = 101: 2 Player mode.
35422 LD (46808),A Later 90 will be written in the timer_flag via 36519.
35425 RET Return.
