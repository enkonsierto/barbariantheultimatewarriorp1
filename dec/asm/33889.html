33889: Converts note value to period.
Converts a note value to a time period for the beeper.
Adds 12 to the value (transposition) and uses note_table as period table.
INPUT: HL = address of note buffer (melody_note or bass_note).
OUTPUT: H = note period, L = 1, DE = note value + 12 (Octave).
DESTROYS: A.
note_to_period_converter 33889 LD A,(HL) Reads note value from buffer.
33890 ADD A,12 Adds 12 (octave/transposition adjustment).
33892 LD E,A E = adjusted note value.
33893 LD D,0 D = 0.
33895 LD HL,34052 HL = base address of period table note_table.
33898 ADD HL,DE HL = table + note index.
33899 LD H,(HL) H = period from table.
33900 LD L,1 L = initial half-cycle counter value.
33902 RET Returns with period in HL.
