37057: Generate left-edge column mask for partial tile blending.
Called from left_edge_renderer for non-0xFF tile bytes at left tile boundary.
Counts leading 1-bits from bit 7 (MSB) to find first transparent column, loads MSB-packed mask (128,192,224,240,248,252,254) into A and falls through to mask_and_blend.
INPUT: A = left-edge tile byte, HL = framebuffer address, DE = tile source.
OUTPUT: A = pixel mask or 0 (fully transparent). DESTROYS: A, C.
left_mask_gen 37057 LD C,A C = A (left-edge tile byte).
37058 BIT 7,C BIT 7,C: if bit 7 clear, byte is #00 fully transparent - jump mask_byte_clear.
37060 JR Z,mask_byte_clear
37062 LD A,128 A = 0x80 (10000000b) test bit 6.
37064 BIT 6,C if bit 6 clear: 1 visible column - jump mask_and_blend with mask 0x80.
37066 JR Z,mask_and_blend
37068 LD A,192 A = 0xC0 (11000000b) test bit 5.
37070 BIT 5,C if bit 5 clear: 2 visible columns - jump mask_and_blend with mask 0xC0.
37072 JR Z,mask_and_blend
37074 LD A,224 A = 0xE0 (11100000b) test bit 4.
37076 BIT 4,C if bit 4 clear: 3 visible columns - jump mask_and_blend with mask 0xE0.
37078 JR Z,mask_and_blend
37080 LD A,240 A = 0xF0 (11110000b) test bit 3.
37082 BIT 3,C if bit 3 clear: 4 visible columns - jump mask_and_blend with mask 0xF0.
37084 JR Z,mask_and_blend
37086 LD A,248 A = 0xF8 (11111000b) test bit 2.
37088 BIT 2,C if bit 2 clear: 5 visible columns - jump mask_and_blend with mask 0xF8.
37090 JR Z,mask_and_blend
37092 LD A,252 A = 0xFC (11111100b) test bit 1.
37094 BIT 1,C if bit 1 clear: 6 visible columns - jump mask_and_blend with mask 0xFC.
37096 JR Z,mask_and_blend
37098 LD A,254 all bits 7..1 set: 7 visible columns, A = 0xFE (11111110b) fall through to mask_and_blend.
