43772: Barbarian Spritesheet
Frame data structure: 1-byte shape ID + 12 tile indices (4 rows x 3 tiles wide).
Shape table sprites_shape_table defines sprite shapes; tile indices reference barb_sprite_tileset (barbarian tileset).
Rendered by draws_4_x_3_tiles (4x3 grid blitter): loads column offset byte, then 12 tiles via draw_single_tile.
P2 uses identical base sprites as P1 with per-frame overlay tiles composited via p2_overlay_tile.
p2_overlay_tile_table: tile index per P2 frame (0-52, 255=no overlay). p2_overlay_pos_table: position bits 0-5 scanline offset from row 94, bits 6-7 column offset (0-3 right of P2 X tile, inverted if mirrored).
Overlay state p2_overlay_block_state mirrors head block head_state layout, using shared renderer render_head_tile + head_mirroring orientation check.
