35910: Key state tester
Used by the routines at entry_point, pause_handler_routine, human_input_reader, main_menu, one_player_mode and two_player_mode.
Converts the scancode and returns if the key is pressed or not in the carry flag.
INPUT: A the scancode.
OUTPUT: Current key state in the carry flag (1 released, 0 pressed).
DESTROYS: A, BC, carry.
key_state_test 35910 LD C,A C scancode to test.
35911 AND 7 Extract row index bits 5-3 add 1 to get 1-based row number in A and B.
35913 INC A
35914 LD B,A B row number used as shift counter.
35915 SRL C Shift row bits into the low position to isolate the row index in A.
35917 SRL C
35919 SRL C
35921 LD A,5 Extract column index bits 2-0 into A.
35923 SUB C
35924 LD C,A C column index bit position within the row byte.
35925 LD A,254 Form keyboard port high byte for this row 0xFE base.
key_state_test_0 35927 RRCA Shift row mask B times to select the correct keyboard half-row port address.
35928 DJNZ key_state_test_0
35930 IN A,(254) Read 5-key state from the ULA keyboard port into A.
key_state_test_1 35932 RRA Shift right C times to place the target key bit into the carry flag.
35933 DEC C
35934 JR NZ,key_state_test_1
35936 RET Returns with carry set key released, carry clear key pressed.
