37218: Composite tile renderer dispatcher.
Used by the routine at tile_render_engine.
Dispatches sub-tiles from a definition sequence for complex sprites that span multiple simple tiles.
Reads control byte from DE (bit7=1 marks composite); indexes table at composite_tile_pointers for sequence pointer in BC at 48269.
Each sequence byte bits7-6 select renderer (11=AND-masked, 00=direct, 01/10=edge); bits4-0=scanlines for B.
Original mirror bit6 XORs 01/10 modes to swap edge renderers. Bit5=1 on next byte ends sequence.
INPUT: A = control byte (bit7=1, bit6=mirror, bits5-0=index 0-63). HL = framebuffer address. DE = tile pointer.
OUTPUT: None. DESTROYS: AF,AF',BC.
Setup: extracts index from control byte and loads definition sequence pointer.
