38664: Step each attribute ink colour one level toward a target.
Called from rainbow_animation each frame to drive the menu rainbow colour animation.
For each of B tiles in the row, reads the ink bits via AND 7. If already at the target value A, leaves the tile unchanged. Otherwise increments the ink by 1, sets BRIGHT, and writes the result back. After all B tiles, advances HL by 32 bytes to the next attribute row and decrements C. If C>1, the outer PUSHes accumulate on the stack but are only popped once on exit, corrupting the caller's stack.
The caller relies on DE=32 on return to step between rows via ADD HL,DE.
INPUT: HL = first attribute address in the row. A = target ink value (0-7). B = tiles per row. C = row count (always 1 in game).
OUTPUT: DE = 32 (row stride; used by caller via ADD HL,DE to advance between rows).
DESTROYS: None (HL, BC, AF all restored via PUSH/POP).
