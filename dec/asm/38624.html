38624: Clear attributes for a row.
Used by the routine at sweep_transition.
Modify colour attributes with customizable width via B and height via C. Used in this case to clear attibutes to a complete row.
Writes A to B consecutive attribute bytes starting at HL, then steps down one attribute row (32 bytes) and repeats for C rows total. Produces a solid colour fill across the given area.
Called from sweep_transition during the fade-out sequence to progressively blacken attribute rows.
INPUT: HL = top-left tile attribute address. B = tiles per row (28). C = row count (01). A = attribute value to write (0 for black).
OUTPUT: None. DESTROYS: C, DE, HL.
