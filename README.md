Barbarian the Ultimate Warrior P1 disassembly
=============================================

A disassembly of the [Spectrum](https://en.wikipedia.org/wiki/ZX_Spectrum) game
[Barbarian the Ultimate Warrior P1](https://en.wikipedia.org/wiki/Barbarian:_The_Ultimate_Warrior).

Created using [SkoolKit](https://skoolkit.ca) and [SkoolKit-Scaffolding](https://github.com/pobtastic/skoolkit-scaffolding).

Browse the latest release:

* [Barbarian the Ultimate Warrior P1 disassembly](https://enkonsierto.github.io/barbariantheultimatewarriorp1/) (hexadecimal)
* [Barbarian the Ultimate Warrior P1 disassembly](https://enkonsierto.github.io/barbariantheultimatewarriorp1/dec/) (decimal)

To build the current development version, first either install SkoolKit, or
make a clone of the [SkoolKit repository](https://github.com/skoolkid/skoolkit)
and point the environment variable SKOOLKIT_HOME at it. Then:

    $ utils/mkhtml.py
