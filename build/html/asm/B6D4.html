B6D4: Attacker state block pointer.
2-byte pointer to the attacker's character state block.
Written by collision_solver with IX (attacker block: p1_controller_type=P1, p2_controller_type=P2).
Read by hit_processor, which loads IY from this address to identify which character delivered the hit, used for score attribution (IYl 89 vs 98). Together with pending_sound_index, forms the combat event descriptor set by collision_solver.
attacker_block_ptr B6D4 DEFW $0000 Pointer to attacker state block. p1_controller_type=P1 attacker, p2_controller_type=P2 attacker.
