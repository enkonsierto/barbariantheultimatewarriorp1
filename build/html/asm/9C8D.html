9C8D: Apply idle movement and toggle facing direction.
Used by the routine at tackle_side_swap.
Helper called twice by tackle_side_swap phase 2 after the side-swap animation completes, once for each fighter.
Applies movement 0 (idle/stop) to reset the animation state, then flips bit 0 of the mirror flag to reverse the character's facing direction.
The caller uses the returned Z flag to select the correct X tile correction: Z (now facing right) adds 2; NZ (now facing left) subtracts 2.
INPUT: IY = character state block. OUTPUT: Z = set if now facing right, NZ if now facing left. DESTROYS: A.
stop_mov_and_mirror 9C8D XOR A Select movement 0 (idle).
9C8E CALL set_char_moves Apply idle movement to the character via set_char_moves.
9C91 LD A,(IY+$02) Load the mirror and control flags byte.
9C94 XOR $01 Toggle the facing direction bit.
9C96 LD (IY+$02),A Write the updated flags byte back.
9C99 BIT 0,A Isolate the new facing bit to set Z or NZ.
9C9B RET Return with Z reflecting the new facing direction.
