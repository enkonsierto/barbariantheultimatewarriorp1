93DC: Round timer tick: decrement, refresh HUD and play sound.
Decrements timer_flag by 1, refreshes the timer display via update_time_display and plays the tick sound via sound_round_end.
Called once per loop iteration by p2_death_score_update and p1_death_score_update while draining the remaining time as bonus score.
INPUT: None. OUTPUT: Z set if the timer has reached 0, NZ if time remains. DESTROYS: AF.
timer_tick 93DC DEC A Load timer value from timer_flag and decrement by 1.
93DD LD ($B6D8),A Write updated timer value back to timer_flag.
93E0 PUSH AF Preserve decremented value and flags across the HUD and sound calls.
93E1 CALL update_time_display Refresh the timer display via update_time_display.
93E4 CALL sound_round_end Play timer tick sound via sound_round_end.
93E7 POP AF Restore timer value and flags.
93E8 OR A Set Z if timer reached 0, NZ if time still remains.
93E9 RET Return.
