9539: Move screen pointer up by B scanlines.
Used by the routine at head_animator.
Counterpart to move_scanl_down. Subtracts 32 from HL up to B times, stepping the framebuffer address back one scanline per iteration.
SBC HL,DE requires carry = 0 on entry; not guaranteed by all callers.
INPUT: B number of scanlines. HL = current framebuffer address. OUTPUT: HL = updated address. DESTROYS: A, DE.
