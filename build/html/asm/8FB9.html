8FB9: Reset P2 state block for new round.
Called from entry_point at game start and from end_of_fight_dispatcher/930D at the end of each fight, after the winning sequence completes.
Increments the fight counter then place the P2 at right sid) with full energy and movement 28 (enter the arena), and sets bit 0 of IY+2 so the P2 starts mirrored (facing left, towards player 1).
Symmetric counterpart to reset_p1_state_block which resets player 1 at tile 0, mirror flag cleared (facing right).
reset_p2_state_block 8FB9 LD A,($B9C6) Increment fight counter fight_counter; if 7 skip write.
8FBC INC A
8FBD CP $08
8FBF JR Z,reset_p2_state_block_0
8FC1 LD ($B9C6),A
reset_p2_state_block_0 8FC4 LD IY,$B562 Points P2 state block (p2_controller_type); Sets A=23 (rightmost tile); call pos_energy_reset to reset position and energy.
8FC8 LD A,$17
8FCA CALL pos_energy_reset
8FCD LD A,($B564) Set P2 mirror flag: SET 0 on IY+2 (p2_mirror_flags) so P2 faces left towards P1.
8FD0 SET 0,A
8FD2 LD ($B564),A
8FD5 RET Return.
