8CD5: Word index to address resolver.
Resolves a word index to its character data address from the string pointer table at string_pointers_table.
INPUT: A = word index. HL = VRAM address.
OUTPUT: HL = VRAM address (preserved). DE = word data address. DESTROYS: None.
word_addr_solver 8CD5 PUSH HL Preserves the VRAM address.
8CD6 LD HL,$BE61 Loads the string pointer table base string_pointers_table into HL.
8CD9 CALL solve_pointer Resolves the pointer for index A via solve_pointer; result in HL.
8CDC POP DE Recovers the VRAM address into DE.
8CDD EX DE,HL Swaps registers: HL=VRAM address, DE=word address.
8CDE RET Return.
