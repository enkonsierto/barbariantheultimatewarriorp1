8F62: Menu rainbow colour animation.
Cycles 7 times (112 ms each), incrementing ink tint across 7 menu rows: white -> yellow -> cyan -> green -> magenta -> green -> cyan (pause/quit conditional).
rainbow_animation 8F62 LD B,$07 Set a counter of 7 loops.
rainbow_animation_0 8F64 PUSH BC Preserves the counter.
8F65 CALL waits_112ms Waits 112 ms.
8F68 LD A,$07 Target ink colour: white (7).
8F6A LD HL,$5945 First attribute row base address for this colour step.
8F6D LD BC,$1501 B=21 tiles per row, C=1 row to step.
8F70 CALL step_ink_up Advance ink colour toward the target value via R38664.
8F73 DEC A Target ink colour: yellow (6).
8F74 LD HL,$5986 Set the initial screen address to colour.
8F77 CALL step_ink_up Increments in 1 the attribute ink.
8F7A ADD HL,DE Selects the next line below.
8F7B DEC A Target ink colour: cyan (5).
8F7C CALL step_ink_up Increments in 1 the attribute ink.
8F7F ADD HL,DE Selects the next line below.
8F80 DEC A Target ink colour: green (4).
8F81 CALL step_ink_up Increments in 1 the attribute ink.
8F84 ADD HL,DE Selects the next line below.
8F85 DEC A Target ink colour: magenta (3).
8F86 CALL step_ink_up Increments in 1 the attribute ink.
8F89 ADD HL,DE Selects the next line below.
8F8A INC A Target ink colour: green (4).
8F8B CALL step_ink_up Increments in 1 the attribute ink.
These two attribute writes only fire if quit_scancode quit and pause_scancode pause scancodes are still 0 not yet assigned. In practice both are non-zero during normal gameplay, so these rows are never coloured at runtime. They would only activate before any key assignment has occurred.
