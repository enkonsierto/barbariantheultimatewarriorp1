39258: Snake energy bar renderer.
Used by the routines at entry_point and game_reset.
Blits the current snake animation frame to the HUD energy bar for each character whose draw phase is active (high nibble of IY+7 non-zero).
P1 bar (VRAM 16416): the sprite is mirrored in-place before blitting and restored afterwards, since P1 faces right and needs the flipped orientation.
P2 bar (VRAM 16445): blitted in its natural orientation.
INPUT: None. OUTPUT: None. DESTROYS: IY, AF, BC, DE, HL.
draw_snakes_sprites 39258 LD IY,46425 Select P1 state block. (p1_controller_type)
39262 LD A,(IY+7) Load P1 packed energy/phase byte.
39265 AND 240 Isolate draw phase index.
39267 JR Z,draw_snakes_sprites_0 Phase idle - skip P1 bar update.
39269 CALL update_snake_frame_id Advance draw phase and fetch snake sprite pointer via update_snake_frame_id.
39272 PUSH DE Save sprite pointer for the restore pass.
39273 PUSH DE Save sprite pointer for the blit pass.
39274 LD B,30 Set scanline count for the mirror operation.
39276 CALL mirror_tile_inplace_0 Mirror snake sprite in-place via mirror_tile_inplace_0.
39279 POP DE Restore pointer to the now-mirrored sprite data.
39280 LD HL,16416 Set P1 energy bar VRAM destination.
39283 LD BC,798 Set blit dimensions - 3 bytes wide, 30 scanlines.
39286 CALL copy_rect Blit mirrored sprite to P1 energy bar via copy_rect.
39289 POP DE Restore pointer to the mirrored sprite data.
39290 LD B,30 Set scanline count for the restore pass.
39292 CALL mirror_tile_inplace_0 Un-mirror sprite in-place via mirror_tile_inplace_0, restoring original data for reuse.
draw_snakes_sprites_0 39295 LD IY,46434 Select P2 state block.
39299 LD A,(IY+7) Load P2 packed energy/phase byte.
39302 AND 240 Isolate draw phase index.
39304 RET Z Phase idle - return.
39305 CALL update_snake_frame_id Advance draw phase and fetch snake sprite pointer via update_snake_frame_id.
39308 LD HL,16445 Set P2 energy bar VRAM destination.
39311 LD BC,798 Set blit dimensions - 3 bytes wide, 30 scanlines.
39314 JP copy_rect Tail-call to blit sprite to P2 energy bar via copy_rect.
