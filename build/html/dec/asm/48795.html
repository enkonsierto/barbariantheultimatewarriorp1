48795: Menu String Sequences
Sequences of byte indices into string_pointers_table. Each sequence defines one menu text line built by
concatenating the words at those indices. The byte 255 terminates each sequence.
Used by draw_string (Draws a string line) and main_menu (Main Menu).
select_option 48795 DEFB 5,6,255 "SELECT OPTION"
0_demo 48798 DEFB 0,28,255 "0 DEMO" Set to "START GAME" (25) by one_player_mode and two_player_mode at mode selection.
1_one_player 48801 DEFB 1,20,7,255 "1 ONE PLAYER"
2_two_player 48805 DEFB 2,21,7,255 "2 TWO PLAYER"
