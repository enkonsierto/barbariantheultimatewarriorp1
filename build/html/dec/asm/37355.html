37355: Resolve tile address.
Used by the routines at draw_single_tile, render_head_tile and sprites_mirror_checker.
Resolves tile index A to sprite RAM address DE.
Selects by character type IYl (107=other other_sprites; else barbarian barb_sprite_tileset). Computes DE=state block+(A*64); each tile=64 bytes (8x8).
INPUT: A = tile index 0-255. IY = character state block. OUTPUT: DE = sprite RAM address.
DESTROYS: AF,DE,HL.
resolve_tile_address 37355 LD D,A Stores tile index in D to form index*256 in DE, ready for the *64 computation via two right-shifts.
37356 LD A,IYl Loads character type IYl to select tile bank. (89)P1, (98)P2 use barb_sprite_tileset. (107)goblin uses other_sprites.
37358 CP 107 If IYl is 107, dispatch to the secondary state block other_sprites.
37360 LD A,D Restores tile index to A.
37361 JR NZ,resolve_tile_address_0 Not goblin - branch to barbarian state block barb_sprite_tileset.
37363 LD DE,40192 Goblin path - load secondary state block base other_sprites.
37366 JR resolve_tile_address_1 Jump to common offset calculation.
resolve_tile_address_0 37368 LD DE,49152 Barbarian path - load barbarian state block base barb_sprite_tileset.
resolve_tile_address_1 37371 PUSH HL Preserve callers HL.
37372 LD H,A Load tile index into H to start index*256 in HL.
37373 LD L,0
37375 SRL H Two right-shifts on HL to obtain index*64 tile byte offset.
37377 RR L
37379 SRL H Add tile byte offset to block base low byte.
37381 RR L Complete 16-bit addition to form final address.
37383 ADD HL,DE DE: final sprite RAM address.
37384 EX DE,HL Restore callers HL.
37385 POP HL
37386 RET Return with DE pointing to tile data.
