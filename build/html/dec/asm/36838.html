36838: Set character movement.
Used by the routines at pos_energy_reset, anim_frame_stepper, init_goblin, post_fight_setup, decap_setup, force_apply_moves, energy_dec, combat_loop, tackle_side_swap and stop_mov_and_mirror.
Changes and stores the active movement ID for the character pointed by IY (reset_p1_state_block or reset_p2_state_block).
Updates the mirror flag, stores the new frame data pointer and stores it.
INPUT: A = new movement ID. IY = character state block.
OUTPUT: IY = character state block with updated mirror flag (IY+2), frame pointer (IY+4, IY+5) initial frame index (IY+8). DESTROYS: AF, BC.
