36022: Null-terminated word renderer.
Used by the routines at draw_string and main_menu.
Draws a null-terminated string from DE to screen at HL, one character per tile.
Called by R36037 and R36130 to render individual words within a menu line.
INPUT: DE = character data address (null-terminated). HL = VRAM destination address.
OUTPUT: HL = VRAM address of the column after the last drawn character. DE: points past the null terminator.
DESTROYS AF, BC, DE, HL.
draw_word 36022 LD A,(DE) Reads the first character byte into A.
draw_word_0 36023 PUSH DE Preserves DE and HL before the draw call.
36024 PUSH HL
36025 CALL draw_character Draws the character via draw_character.
36028 POP HL Restores HL and DE.
36029 POP DE
36030 INC L Advances HL to the next column and DE to the next character.
36031 INC DE
36032 LD A,(DE) Reads the next character byte.
36033 OR A Tests for null terminator. (0 = end of word).
36034 JR NZ,draw_word_0 Loops back if more characters remain.
36036 RET Return.
