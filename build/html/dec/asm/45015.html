45015: Hit section pointer tables. Dispatch indexed by attacker frame.
Each sub-table is a list of 2-byte pairs (defender_frame, hit_type_index) terminated by 0xFF. hit_type_index is an index into hit_type_table.
Used by hit_check to test if IY+8 (defender frame) is vulnerable to the current attacker frame IX+8.
