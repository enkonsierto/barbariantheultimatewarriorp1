47440: P2 keyboard scancode table.
5-byte table: Fire, Up, Down, Left, Right scancodes for P2 (controller type 2).
Pre-loaded with this keys: 35,27,19,3,11.
Written by key_assign key-assignment loop when P2 is assigned a custom keyboard
(control_setup sets IX=p2_keyboard_table). Used in 2P options 2 and 3.
Read by human_input_reader each frame when P2 controller type is 2.
p2_keyboard_table 47440 DEFB 35 Fire = key 0
47441 DEFB 27 Up = key 9
47442 DEFB 19 Down = key 8
47443 DEFB 3 Left = key 6
47444 DEFB 11 Right = key 7
