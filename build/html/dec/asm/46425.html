46425: Player 1 state block.
9-byte structure used by all combat routines.
Written by pos_energy_reset (init), set_char_moves (movement change), energy_dec (energy), input_ai_dispatcher (input).
IY+0 controller type: 1=P1 keyboard p1_keyboard_table, 2=P2 keyboard p2_keyboard_table (Sinclair 2 defaults), 3=Sinclair2 fixed sinclair_joystick_2_table (keys 6-0), 4=Sinclair1 fixed sinclair_joystick_1_table (keys 1-5), 5=Kempston port 31. Bit 7 set=CPU-controlled.
Character type is implicitly IYl (low byte of IY base address: 89=P1, 98=P2, 107=goblin).
p1_controller_type 46425 DEFB 1 IY+0: Controller type (0=inactive, 1-5=human, bit 7 set=CPU).
p1_input_state 46426 DEFB 0 IY+1: Input state bitmask (bits 0-4: Right, Left, Down, Up, Fire).
p1_mirror_flags 46427 DEFB 0 IY+2: Mirror and priority flags (bit 0=facing left, bit 7=draw priority).
p1_x_pos 46428 DEFB 4 IY+3: X tile position (0-29).
p1_frame_ptr 46429 DEFW frame_seq IY+4+5: Frame data pointer.
p1_movement_id 46431 DEFB 1 IY+6: Current movement ID (1-37).
p1_energy_byte 46432 DEFB 6 IY+7: Energy nibbles (bits 0-3=current energy 0-13, bits 4-7=display update flag).
p1_frame_index 46433 DEFB 0 IY+8: Current frame index byte (read from frame data pointer).
