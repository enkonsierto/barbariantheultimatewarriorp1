37159: Renders a 3-byte-wide sprite tile over 21 scanlines using AND masking.
Per scanline: saves 3 framebuffer bytes to AND-mask scratch (mask_scratch_buffer), loads mask from tile data via left_edge_renderer, draws sprite pixels via right_edge_renderer, then ANDs with scratch mask for transparency.
Framebuffer/tile pointers use alternate register bank (EXX) so right_edge_renderer auto-advances to next scanline.
INPUT: B = scanlines (21), HL = framebuffer address, DE = tile data pointer.
OUTPUT: None. DESTROYS: AF,BC,DE,HL (and shadows).
tile_renderer 37159 LD B,21 B = 21 (full tile height).
This entry point is used by the routine at comp_tile_dispatcher.
tile_renderer_0 37161 PUSH BC save B (scanlines remaining).
37162 PUSH HL save HL (framebuffer), DE (tile ptr).
37163 PUSH DE
37164 EXX EXX: framebuffer HL/tile DE -> alternate registers.
37165 POP DE DE = tile data (recovered).
37166 POP HL HL = framebuffer (recovered).

37168 LD A,(HL) save 3 background bytes to AND-mask scratch (mask_scratch_buffer).

37182 DEC L DEC L x2: restore HL to start of 3-byte sequence
37183 DEC L
37184 EXX EXX: back to main registers. HL'/DE' hold framebuffer/tile pointers.
37185 LD HL,47455 HL = mask scratch buffer (mask_scratch_buffer).
37188 CALL left_edge_renderer left_edge_renderer: load AND mask from tile data into scratch buffer.
37191 EXX EXX: activate HL=framebuffer, DE=tile for right_edge_renderer.
37192 CALL right_edge_renderer right_edge_renderer: draw one scanline of sprite pixels to framebuffer.
37195 EXX EXX: back to main registers (now point to next scanline).
37196 LD HL,47455 HL = scratch buffer (reload).
37199 POP DE DE = framebuffer address of the scanline just rendered.
37200 LD A,(DE) framebuffer byte 0 (w/sprite); AND scratch byte 0 (mask); restore transparent areas
37201 AND (HL)
37202 LD (DE),A

37205 LD A,(DE) framebuffer byte 1; AND scratch byte 1; restore transparent areas

37210 LD A,(DE) framebuffer byte 2; AND scratch byte 2; restore transparent areas

37212 LD (DE),A

37215 DJNZ tile_renderer_0 copied one scanline. Repeat for all scanlines
